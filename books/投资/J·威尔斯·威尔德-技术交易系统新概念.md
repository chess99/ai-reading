# 技术交易系统新概念

## 书籍概述

《技术交易系统新概念》是技术分析领域的经典之作，由J·威尔斯·怀尔德（J. Welles Wilder, Jr.）于1978年出版。本书原创性地提出了一系列至今仍被广泛使用的技术指标和交易系统，包括**抛物线转向系统（Parabolic SAR）**、**相对强弱指数（RSI）**、**方向运动指数（DMI/ADX）**、**波动率指数**以及**商品选择指数（CSI）**等。

怀尔德的核心观点是：**没有任何单一的交易系统能在所有市场中持续盈利**。因此，他的解决方案是建立一个评级体系，让交易者能够在正确的时机选择正确的市场进行交易。这一理念贯穿全书。

## 第二部分：抛物线时间/价格系统（Parabolic Time/Price System）

### 核心原理

抛物线系统是**真正的反转系统**，每次止损点同时也是反向入场点。系统名称来源于：当价格在图表上绘制时，止损点形成的图案类似抛物线（法国曲线）。

**时间/价格概念**：系统允许市场在交易初期有所反应，然后止损开始加速移动。止损不仅取决于价格，也取决于时间。**止损永不后退**——它每天只沿开仓方向递增。

### 计算公式

```
SAR明天 = SAR今天 + AF × (EP交易 - SAR今天)
```

其中：
- **AF**（Acceleration Factor，加速因子）起始值为0.02
- 每次创出新有利价位时，AF增加0.02
- AF上限为0.20
- **EP**（Extreme Price，极端价格）：做多时为期间最高价，做空时为期间最低价

### 具体计算示例

假设第4天从空头反转为多头：

```
第4天：SAR = 50.00（前一空头交易的最低SIP）

第5天计算：
SAR₅ = 50.00 + 0.02 × (52.50 - 50.00) = 50.00 + 0.05 = 50.05

第6天（无新高）：
SAR₆ = 50.05 + 0.02 × (52.50 - 50.05) = 50.05 + 0.05 = 50.10

第8天（有新高52.50，AF增至0.04）：
SAR₈ = 50.19 + 0.04 × (52.50 - 50.19) = 50.19 + 0.09 = 50.28
```

### 重要规则

1. **第一天的SAR是前一重要点（SIP）**
2. **止损不能移入前一天或当天的价格区间**
3. AF增量范围：**0.018至0.021**（任何在此范围内的增量效果相近）
4. AF上限为0.20，不超过0.22

### 系统应用建议

- 主要用于**有方向性趋势的市场**
- 可结合**方向运动指数**使用：方向向上时只做多，向下时只做空
- 在趋势明确的市场中表现优异

## 第三部分：波动率指数与波动率系统

### 真波幅（True Range）

怀尔德定义的**真波幅**是以下三个值的最大者：

```
TR = max(Hₜ - Lₜ, Hₜ - Cₜ₋₁, Cₜ₋₁ - Lₜ)
```

**注意**：限价日（涨停或跌停）的真波幅是从前一收盘价到限价的距离。

### 波动率指数（Volatility Index）

波动率指数用于衡量市场波动性：

```
VI今天 = (13 × VI前值 + TRₜ) / 14
```

经过大量测试，怀尔德发现**14天**能最好地反映波动性。

### 波动率系统

**真均值波幅（ATR）**：基于7天真波幅的平均值

```
初始ATR = Σ(前7天TR) / 7

后续ATR = (6 × ATR前值 + TR今天) / 7
```

**ARC（平均波幅常数）** = ATR × C（常数3.0，范围2.8-3.1）

### 波动率系统规则

| 持仓方向 | 入场/出场 | 规则 |
|---------|---------|------|
| 多头 | 止损反转点 | 最高收盘价 - ARC |
| 空头 | 止损反转点 | 最低收盘价 + ARC |

**特点**：
- 止损与极端收盘价相关，同时与波动率相关
- 波动率增加时，止损自动远离价格
- 波动率降低时，止损自动靠近价格
- 这是**真正的反转系统**

## 第四部分：方向运动概念（核心创新）

这是本书**最重要的概念**。怀尔德成功将"方向运动"这一模糊概念转化为精确的数学方程。

### 方向运动的定义

**方向运动的基本增量**：今日价格区间中，**昨日区间之外的最大部分**

```
PLUS DM (+DM)：区间向上超出昨日
MINUS DM (-DM)：区间向下超出昨日
```

### 各种价格形态的处理

| 形态 | 处理方式 |
|------|---------|
| **外部日**（今天区间完全超出昨天） | 只取较大的DM，忽略较小的 |
| **内部日**（今天区间在昨天内部） | DM = 0 |
| **极限日** | 计算从昨日收盘到极限价位的距离 |

### 方向指标（Directional Indicator）

为了使DM有意义，必须将其表示为波幅的函数：

```
+D1 = +DM / TR
-D1 = -DM / TR
```

### 14日方向运动指数

使用**累积技术**计算：

```
今日+DM₁₄ = 前日+DM₁₄ - 前日+DM₁₄/14 + 今日+DM

今日-DM₁₄ = 前日-DM₁₄ - 前日-DM₁₄/14 + 今日-DM

今日TR₁₄ = 前日TR₁₄ - 前日TR₁₄/14 + 今日TR
```

### 方向运动指数（DX）

```
+DI₁₄ = +DM₁₄ / TR₁₄ × 100
-DI₁₄ = -DM₁₄ / TR₁₄ × 100

DX = (|+DI₁₄ - -DI₁₄|) / (+DI₁₄ + -DI₁₄) × 100
```

**DX值范围**：0到100
- **DX越高**：方向性运动越强
- **DX越低**：市场越无方向性（震荡）

### 平均方向运动指数（ADX）

使用14天DX的移动平均：

```
第一ADX = Σ(前14天DX) / 14

后续ADX = (前日ADX × 13 + 今日DX) / 14
```

### 方向运动指数评级（ADXR）

```
ADXR = (ADX今天 + ADX₁₄天前) / 2
```

**ADXR用途**：
- 用于**商品选择指数（CSI）**的评级
- ADXR > 25：适合使用趋势跟踪系统
- ADXR < 20：不建议使用趋势跟踪系统

### 方向运动系统规则

| 信号 | 条件 |
|------|------|
| **做多入场** | +DI₁₄从下向上穿越-DI₁₄ |
| **做空入场** | -DI₁₄从上向下穿越+DI₁₄ |
| **极端点规则** | 交叉日当天极端价位作为止损点 |

### ADX图表解读

- 当ADX线从下方穿越两条DI线时，产生转向信号
- ADX在两条DI线上方**转头向下**时，通常是获利了结时机
- ADX跌至两条DI线下方时，应停止使用趋势跟踪系统

## 第五部分：动量概念与趋势平衡点系统

### 动量因子（Momentum Factor）

```
MF = 今日收盘价 - 前天收盘价
```

**解读**：
- 连续多天价格上涨但幅度递减 → 动量为负（减速）
- 连续多天价格上涨且幅度递增 → 动量为正（加速）

### 趋势平衡点（Trend Balance Point, TBP）

**TBP定义**：
- 做多时：收盘价必须**低于**TBP才会反转为空头
- 做空时：收盘价必须**高于**TBP才会反转为多头

**TBP计算**：

```
做多时：选择前两个MF中较低的，加上昨日收盘价
做空时：选择前两个MF中较高的，加上昨日收盘价
```

### 趋势平衡点系统规则

| 条件 | 操作 |
|------|------|
| MF今日 > 前两日MF | 做多（在收盘价高于TBP时） |
| MF今日 < 前两日MF | 做空（在收盘价低于TBP时） |

**止损**：X̄ ± TR（X̄ = (H+L+C)/3，TR = 真波幅）

**目标**：
```
做多目标 = 2X̄ - L（最低价）
做空目标 = 2X̄ - H（最高价）
```

**系统特点**：
- 目标相对较近，止损较远（与传统系统相反）
- 正确率可达70%-80%
- 适合需要频繁交易的交易者

## 第六部分：相对强弱指数（RSI）

RSI是本书最著名的指标，至今仍是技术分析中使用最广泛的振荡器之一。

### RSI公式

```
RS = 平均上涨收盘价 / 平均下跌收盘价

RSI = 100 - 100 / (1 + RS)
```

**默认参数**：14天

### 初始计算

1. 计算前14天所有上涨日的平均涨幅
2. 计算前14天所有下跌日的平均跌幅
3. 计算RS和RSI

### 后续计算（累积法）

```
新平均上涨 = (前日平均上涨 × 13 + 今日上涨) / 14
新平均下跌 = (前日平均下跌 × 13 + 今日下跌) / 14
```

### RSI解读

| RSI值 | 含义 |
|-------|------|
| **> 70** | 超买，可能顶部 |
| **< 30** | 超卖，可能底部 |
| **< 30后转向上** | 潜在买入信号 |
| **> 70后转向下** | 潜在卖出信号 |

### RSI的五大应用

1. **顶部和底部**：RSI突破70或跌破30先于价格见顶/见底
2. **图表形态**：RSI上的头肩顶/底、三角形等形态可能先于价格出现
3. **失败摆动（Failure Swing）**：在70以上或30以下形成的失败摆动是强烈反转信号
4. **支撑和阻力**：RSI上的支撑/阻力线可确认价格图表的趋势线
5. **背离**：价格创新高/低但RSI未跟随，是最强烈的反转信号

## 第七部分：反应趋势系统（Reaction Trend System）

这是一个**混合系统**，结合了反趋势和趋势跟踪两种模式。

### 两种模式

| 模式 | 描述 | 适用市场 |
|------|------|---------|
| **反应模式** | 逢低买入，逢高卖出 | 震荡/无趋势市场 |
| **趋势模式** | 追踪突破，持有趋势 | 快速趋势市场 |

### 四个价格作用点

基于X̄ = (H+L+C)/3计算：

```
B₁（买入点）= 2X̄ - H
S₁（卖出点）= 2X̄ - L
HBOP（向上突破点）= 2X̄ - 2L + H
LBOP（向下突破点）= 2X̄ - 2H + L
```

### B-O-S定相序列

将交易日分为三类循环：
- **B日**：只能做多
- **O日**：无新仓（除非触发突破）
- **S日**：只能做空

### 规则摘要

**反应模式**：
- 只在B日做多（在B₁买入）
- 只在S日做空（在S₁卖出）
- 目标为S₁（做多）或B₁（做空）

**趋势模式**：
- 价格突破HBOP或LBOP时自动进入
- 使用**追踪止损**：前两天最高/最低价
- 触发止损后退出，不反方向开仓

### 适用市场

系统在**低方向运动指数**（ADXR较低）的市场中表现优异，这些市场通常让其他趋势跟踪系统频繁被扫损。

## 第八部分：摆动指数（Swing Index）

这是本书技术性最强的部分，由怀尔德与合作者开发。

### 摆动指数公式

```
SI = 50 × [(C₂-C₁) + 0.5(C₂-O₂) + 0.25(C₁-O₁)] / K × R/L
```

其中：
- **K** = max(H₂-C₁, L₂-C₁) 的绝对值
- **R** = 根据三种情况计算的复杂函数
- **L** = 当日价格限制值

### 累积摆动指数（ASI）

```
ASIₜ = ASIₜ₋₁ + SIₜ
```

ASI将每日的SI值累加，形成一条连续曲线。

### 摆动指数特性

1. **值域**：-100到+100
2. **识别摆动**：自动识别短期价格摆动点
3. **抑制噪音**：日内大幅波动不直接影响指数（权重偏向收盘价）

### 摆动指数系统

**入场信号**：
- 做多：ASI超越前一显著**高点摆动点（HSP）**时的ASI值
- 做空：ASI跌破前一显著**低点摆动点（LSP）**时的ASI值

**止损反转规则**：
- 做多时：前一LSP作为初始止损
- 做空时：前一HSP作为初始止损

**追踪止损**：
- 60 ASI点的追踪止损
- 在有利方向移动后跟踪止损位置

## 第九部分：商品选择指数（CSI）

CSI是怀尔德理论的**最终整合**，解决了"交易什么"的问题。

### CSI公式

```
CSI = ADXR × ATR₁₄ × K

其中：
K = 50/L × √(V/M) × 100/(150+C)

L = 跳动值（以美元计）
V = 1点价值（美元）
M = 保证金要求（美元）
C = 佣金（美元）
```

### CSI考虑的因素

| 因素 | 权重 | 来源 |
|------|------|------|
| 方向运动 | 高 | ADXR |
| 波动性 | 高 | ATR₁₄ |
| 保证金要求 | 中 | M |
| 佣金成本 | 低 | C |

### CSI应用

- 每天计算所有关注商品的CSI
- **只交易CSI最高的5-6个商品**
- 当商品从CSI榜单上消失时，退出该仓位

### 怀尔德的观点

> 大多数商品只有**约30%的时间**处于良好趋势中（高方向运动）。如果交易者始终关注相同商品，他的系统必须足够好，能在30%的时间赚钱以弥补70%时间的亏损。相比之下，只交易CSI榜单前几名的方法更加高效。

## 第十部分：资金管理

### 怀尔德的资金管理原则

1. **单品种保证金限制**：任何单一商品上的保证金不超过总资金的**15%**
2. **总保证金限制**：任何时候的总保证金不超过总资金的**60%**

### 怀尔德的推荐做法

- 交易CSI榜单前6名商品
- 每个商品使用不超过总资金**10%**的保证金
- 专注于资本保护

### 亏损恢复表

| 亏损% | 恢复所需盈利% |
|-------|--------------|
| 5 | 5.3 |
| 15 | 17.6 |
| 30 | 42.9 |
| 50 | 100.0 |
| 70 | 233.0 |

**核心洞察**：亏损越大，恢复难度呈几何级数增长。

## 核心理念总结

```
┌─────────────────────────────────────────────────────────────┐
│                    完整交易计划的三个组成部分                  │
├─────────────────────────────────────────────────────────────┤
│  1. 良好的技术交易系统                                       │
│  2. 在正确的时机、正确的市场使用该系统                          │
│  3. 良好的资金管理技术                                        │
├─────────────────────────────────────────────────────────────┤
│  重要性排序：资金管理 > 系统适配 > 交易系统                     │
└─────────────────────────────────────────────────────────────┘
```

怀尔德的核心贡献在于：**将主观的"趋势"概念客观化、定量化**。通过方向运动指数，交易者可以客观判断市场是否处于趋势状态，从而选择适当的交易策略。
