# 日本蜡烛图技术

## 概述：蜡烛图的优势

蜡烛图是日本最流行、最古老的技术分析形式，其历史早于西方条形图和点线图数百年。与条形图相比，蜡烛图具有以下独特优势：

1. **直观呈现多空力量**：通过实体颜色和影线长度，形象反映市场供需状况
2. **揭示行情背后的力量**：不仅显示趋势，还揭示推动趋势变化的多空博弈
3. **信号更及时**：蜡烛图往往在1～3个交易日内发出反转信号，而条形图可能需要数周

蜡烛图使用的数据与条形图相同（开盘价、最高价、最低价、收盘价），但提供了条形图无法呈现的丰富信息——这是一种“双赢”工具。

## 第二部分：单根蜡烛线形态

### 1. 长白实体与长黑实体

**长白实体**的判断标准：
- 开盘于最低价附近，收盘于最高价附近
- 实体长度至少是前一天实体的3倍
- 代表多方完全掌控局面

**长白实体的三大应用**：

**① 低价区构成支撑**
- 长白实体出现于低价区可能是底部第一个信号
- 其低点构成支撑区域，精确位置在实体的中点至低点

**② 突破阻力验证**
- 长白实体穿越阻力位，证明市场势头强劲
- 突破后留下的缺口往往成为后续重要支撑

**③ 确认既有支撑**
- 由支撑位向上反弹的长白实体，显示多头强劲
- 支撑线与蜡烛图支撑位可相互验证

**长黑实体**的判断标准：
- 实体长度明显大于先前数根蜡烛线
- 收盘价接近最低价，开盘价接近最高价

**长黑实体的三大应用**：
- 高价区出现可能是顶部信号
- 确认阻力位的有效性
- 向下突破支撑时表明卖压沉重

**实战法则**：长白实体的中点可视为潜在支撑位；长黑实体的中点可视为潜在阻力位。

### 2. 开盘价与前一实体关系的分析

**激进交易技巧**：根据开盘价位置判断当日多空态势，无需等待收盘。

- **空头信号**：开盘价在前一根白线实体的中点以下
- **多头信号**：开盘价在前一根黑线实体的中点以上

这种方法在股票市场比期货市场更适用，因为期货价格波动更剧烈，开盘价更可能远离前日收盘价。

### 3. 纺锤线

**定义**：实体很小的蜡烛线（可黑可白），代表多空双方处于拉锯状态。

**特征**：
- 上下影线可能很长
- 市场如同“喘息困难”

**重要应用：筹码积聚与派发**

| 类型 | 发生位置 | 成交量特征 | 市场含义 |
|------|---------|-----------|---------|
| **积聚** | 低价区 | 成交量放大，价格呆滞 | 空头打光子弹后被多头接手，预示上涨 |
| **派发** | 高价区 | 成交量放大，价格停滞 | 聪明钱正在派发筹码给买家，预示见顶 |

**识别要点**：纺锤线配合大成交量出现时，如果处于低价区且价格曾创出新低但无力守住，这是积聚信号（看涨）；如果在高价区且长白实体后出现放量纺锤，这是派发信号（看跌）。

### 4. 十字星

**定义**：开盘价与收盘价相等或几乎相等的蜡烛线，没有实体或几乎没有实体。

**十字星的核心原则**：
- 十字星表示市场处于“十字路口”和犹豫状态
- **不应直接将十字星视为买进或卖出信号**
- 应等待1～2个交易时段观察市场选择方向

**十字星的四大判断规则**：

```
┌────────────────────────────────────────────────────────┐
│  十字星出现位置          │  市场含义                   │
├────────────────────────────────────────────────────────┤
│  ① 急升或超买之后        │  可能见顶信号（最重要）     │
│  ② 上升初期              │  构成顶部机会较小           │
│  ③ 大跌之后              │  可能见底信号               │
│  ④ 下跌刚开始            │  行情可能继续下跌           │
└────────────────────────────────────────────────────────┘
```

**超买或超卖越严重，十字星作为反转信号的参考价值越大。**

**十字星的延伸应用**：
- **构成阻力**：十字星的顶端经常意味着阻力位
- **恢复信号**：如果十字星顶点被超越，表示市场“从疲劳中恢复”，可作为买入确认
- **墓碑十字星**：开盘价、收盘价与最低价处于同一位置的十字星，类似墓碑，出现在高价区尤其危险

### 5. 高浪线

**定义**：上下影线都很长的蜡烛线，代表多空对峙，市场“丧失方向感”。

**判断要点**：高浪线出现在上升或下降趋势后，预示先前趋势可能岌岌可危。

**高浪十字星**：同时带有很长上下影线的十字星，也称“长腿十字星”。

## 第三部分：单蜡烛线反转形态

### 锤子线（Hammer）

**形态特征**：
```
        │
    ┌───┴───┐
    │       │
    │   ■   │  ← 小实体（可黑可白）位于上端
    │       │
    └───────┘
        │
```

**四个判定标准**：
1. 此前趋势必须向下（大幅下跌或严重超卖）
2. 实体很小且处于价格区间上端
3. 下影线长度为实体3倍或以上
4. 上影线很短或没有

**交易策略**：
- **激进做法**：在锤子线出现后立即少量做多试水
- **保守做法**：等待价格回落测试锤子线支撑，确认支撑有效后再加码
- **止损位**：锤子线低点的下方（以收盘价为准）

**关键理解**：锤子线必须出现在大幅下跌之后才有意义。在横盘或小幅下跌后出现的类似形态，不是真正的锤子线。

### 吊颈线（Hanging Man）

**形态特征**：与锤子线完全相同，但位置相反。

**日本人的经典比喻**：
> “如果它出现在下方，买入；如果它出现在上方，卖出。”

**判定关键**：同一形状的蜡烛线，究竟是看空的吊颈线还是看多的锤子线，完全取决于先前的趋势。

**确认原则**：吊颈线需要等待下一个交易日确认——收盘价必须低于吊颈线实体，才构成走熊确认。

**为什么需要确认？**：
- 长下影线表示买盘意愿存在
- 但收盘价跌破实体意味着在开盘价与收盘价附近买进的人都在亏损
- 这些多头可能被迫平仓，导致价格进一步走弱

### 流星线（Shooting Star）

**形态特征**：
```
        │
        │  ← 长上影线（核心特征）
    ┌───┴───┐
    │   □   │  ← 小实体（可黑可白）位于下端
    └───────┘
```

**判定标准**：
1. 必须出现在明确的上升趋势之后
2. 上影线很长，代表市场拒绝接受新高价格
3. 实体很小且接近蜡烛线下端

**交易含义**：
- 空头有能力让价格从高位急挫而下
- 是空头反转信号

### 三者对比总结

| 形态 | 之前趋势 | 影线特征 | 实体位置 | 信号方向 |
|------|---------|---------|---------|---------|
| **锤子线** | 下降 | 长下影线 | 上端 | 看涨 |
| **吊颈线** | 上升 | 长下影线 | 上端 | 看跌（需确认） |
| **流星线** | 上升 | 长上影线 | 下端 | 看跌 |

## 第四部分：双蜡烛线形态

### 1. 乌云盖顶（Dark Cloud Cover）

**形态结构**：
```
    第一根：强劲长白实体
    第二根：黑实体，开盘高于第一根收盘，收盘切入第一根实体深处
```

**判断标准**：
- 标准形态：第二根黑线收盘价应切入第一根白线实体的**中点以下**
- 切入越深，空头意味越强烈

**股市灵活标准**：
- 期货市场：第二根开盘价应高于第一根高点
- 股票市场：第二根开盘价只需高于第一根收盘价即可

**重要特性**：乌云盖顶经常成为后续走势的**阻力位**。

### 2. 刺穿形态（Piercing Pattern）

**形态结构**：
```
    第一根：长黑实体
    第二根：白实体，开盘创出新低，收盘切入第一根实体
```

**判断标准**：
- 第二根白线收盘价应切入第一根黑线实体的**一半以上**
- 切入越深，信号越积极

**日本文献的生动描述**：
> “多方最后的力量被逼到角落后跳出来进行悲壮的一搏。这种神风特攻队员的姿态总是非常可怕的，空方只得暂时退避一旁。”

**市场含义**：下行能量已经被消耗殆尽。

### 3. 吞噬形态（Engulfing Pattern）

**多头吞噬**（底部反转）：
```
    第一根：黑实体（可很短）
    第二根：白实体，完全包裹第一根
```

**空头吞噬**（顶部反转）：
```
    第一根：白实体（可很短）
    第二根：黑实体，完全包裹第一根
```

**信号强度判断**：
1. **最强**：第一根实体很短，第二根实体很长且完全包裹（包括影线）
2. **次强**：两根影线都被包裹（第二根高点高于前高，低点低于前低）
3. **重要例外**：如果两根实体长度大致相当，可能只是横盘而非反转

**实战要点**：如果收盘价超越空头吞噬的高点（上影线顶点），盘面由看空转为看多。

### 4. 最后吞噬形态

**最后吞噬底**：外观与空头吞噬相同，但出现在下降走势中，是底部反转信号。

**最后吞噬顶**：外观与多头吞噬相同，但出现在上升走势中，是顶部反转信号。

**日本人的比喻**：
> “双双殉情”——最后的长白实体让你爱上市场，但此后也可能和市场一起毁灭。

### 5. 包孕形态（Harami）

**结构**：
```
    第一根：长实体（可黑可白）
    第二根：小实体，完全处于第一根实体之内
```

**信号强度**：
- 第二根实体位于第一根中点：反转概率较大
- 第二根实体位于第一根上部（上升趋势中）：可能横盘而非下跌
- 第二根为十字星而无实体：反转概率最大（“包孕十字”）

**重要结论**：
- 下跌趋势中的包孕形态代表“下跌动能正趋于衰竭”
- 上升趋势中的包孕形态代表“行情没能守住高位”

**高价包孕与低价包孕**：
- **高价包孕**：第二根小实体靠近第一根实体上端，后市可能横盘而非下跌
- **低价包孕**：第二根小实体靠近第一根实体下端，后市可能横盘而非上涨

**期权交易启示**：高价包孕或低价包孕出现后，行情经常由明确趋势转为横盘（波动率降低），适合卖空波动率策略。

### 6. 窗口（缺口/Gap）

**定义**：
- **上升窗口**：今天蜡烛线的低点高于前一天蜡烛线的高点
- **下降窗口**：今天的高点低于前一天蜡烛线的低点

**窗口的理论基础**：
- 窗口是行情的**中继形态**，出现后趋势将延续
- 上升窗口是**多头持续形态**
- 下降窗口是**空头持续形态**

**日本谚语**：
> “反向走势遇窗口而止”——上升后的回落应在窗口内获得支撑；下跌后的反弹应在窗口内遇阻回落。

**核心确认原则**：
> **如果收盘价封闭窗口，则窗口出现之前的趋势就会结束。**

**实战要点**：
- 必须等待**收盘价**确认窗口是否被封闭
- 盘中跌破不算，必须收盘价跌破
- 对于周线图，须等周收盘价确认

**等待确认原则**：三个交易时段法则——如果窗口在三个交易时段内没有关闭，市场将顺着缺口方向运行。这反映了日本文化对数字"三"的重视。

### 7. 三窗口理论

日本技术分析认为，连续出现三个上升窗口或下降窗口，代表趋势已达极致，物极必反。

**交易策略**：
- 出现三个上升窗口后，应了结多头头寸或做保护性交易
- 但在最近一个窗口被收盘价封闭以前，不宜激进逆势操作

### 8. 双黑跳空

**定义**：下降窗口后连续出现两个黑色实体，是极强的空头信号。

**日本说法**：“多头陷入溃败。”

### 9. 跳空十字星

**定义**：下降走势中出现向下跳空的十字星，通常代表卖压增强。

**建议**：等待确认——如果次日出现长白实体且价格上涨，形成启明星形态，则转为多头信号。

## 第五部分：三蜡烛线及以上形态

### 1. 黄昏星（Evening Star）

**结构**：
```
    第一根：长白实体（上升趋势中）
    第二根：小实体（可白可黑），不触及第一根实体
    第三根：黑实体，深度切入第一根白实体
```

**黄昏十字星**：第二根蜡烛线是十字星而非小实体。

**等待确认原则**：只有第三根黑实体出现并切入第一根白实体后，才可确认走熊。

**《黄金泉》的经典阐述**：
> “当走势发展至极端状态时，就会出现停滞。停滞生阴。”

### 2. 启明星（Morning Star）

**结构**：
```
    第一根：长黑实体（下降趋势中）
    第二根：小实体（可白可黑），不触及第一根实体
    第三根：长白实体，向上切入第一根黑实体
```

**启明十字星**：第二根蜡烛线是十字星。

**弃婴形态**：启明十字星中，如果十字星与前后两根蜡烛线之间都存在跳空缺口（非常罕见）。

### 3. 崩盘十字星（Crasching Doji）

**结构**：
```
    第一根：长白实体
    第二根：向下跳空十字星
    第三根：黑实体，在十字星下方跳空低开
```

**与黄昏十字星的区别**：黄昏十字星的十字星跳空高开在第一根白实体上方；崩盘十字星的十字星跳空低开在第一根白实体下方。

**日本警告**：“大跌的前兆。”

### 4. 迭创新高（低）形态

**核心概念**：如果出现几乎连续的8～10个新高蜡烛线（或新低蜡烛线），意味着先前趋势面临改变的可能性大大增强。

**日本别称**：“酒田身体里的根根骨头”——凸显其作为技术分析基础的重要性。

**计数方法**：
- 找到此前走势中的最低点（或最高点）
- 创出新低后第二个交易日的高点抬高，构成“新低蜡烛线1”
- 此后如有交易日创出更低，构成“新低蜡烛线2”
- 中间的横盘蜡烛线不计入，但不超过2～3根

**交易含义**：
- 出现8～10个新高蜡烛线后，应停止买进（甚至考虑卖空）
- 出现8～10个新低蜡烛线后，应停止卖出（甚至考虑买入）

**重要补充**：市场化解超买/超卖的方式有两种：横向盘整或大幅修正。一旦化解，迭创新高（低）形态就不再成立。

## 第六部分：蜡烛图与整体技术面

### 核心原则：形态的位置比形态本身更重要

孤立地分析蜡烛图形态是危险的。任何蜡烛图信号必须置于市场整体技术形态的大背景中来观察。

**实战案例**：比较乌云盖顶与空头吞噬：
- 孤立看，空头吞噬信号更强
- 但如果乌云盖顶发生在重要阻力位，而空头吞噬不在重要位置，前者的空头反转信号可能更强

### 止损原则

**核心理念**：当价格到达某个点位时，必须承认判断错误，这个价位就是止损点。

**止损设置方法**：
- 根据空头信号做空时，止损点设在形态高点的上方
- 根据多头信号做多时，止损点设在形态低点的下方
- **必须以收盘价确认是否触发止损**

**人性弱点警示**：一旦市场实际与主观判断背道而驰，人性弱点决定有些交易者会一厢情愿地盲目坚持。但一厢情愿的成本总是太高昂。

### 风险与收益的权衡

**核心问题**：某个蜡烛图形态出现时，并不意味着最佳交易时机已来临。

**决策流程**：
1. 识别蜡烛图形态
2. 评估风险与收益比率是否诱人
3. 如果比率不诱人，应放弃交易

**日本警句**：
> “勇士的潜能是一张拉满的弓，但关键在于放弦的时机。”

**设定目标价位的方法**：
- 艾略特波浪理论
- 支撑位与压力位
- 斐波那契回调比例
- 先前蜡烛图形态的密集区

**何时放弃交易**：
- 形态的风险与收益比率不诱人
- 市场处于长阴长阳交错的大幅震荡时期（“像8级地震”）

### 趋势判断与顺势交易

**核心策略**：顺着趋势的方向下单交易，反转信号出现后冲销既有头寸。

```
    上升趋势：出现空头信号 → 平多头或设保护
    上升趋势：出现多头信号 → 加仓或建新多头
    
    下降趋势：出现空头信号 → 加空或建新空头
    下降趋势：出现多头信号 → 平空头
```

**趋势判断工具**：
- 趋势线
- 移动平均线
- 价格高低点的排列

### 做一个市场“变色龙”

**核心理念**：必须持续关注行情演变轨迹，以确定价格变动是否符合预期。如果市场出乎意料，必须调整判断。

**日本策略**：
- 当形态完成后，对行情的期待形成自己的判断
- 确定“如果行情证明我错，价格会到达哪个点位”
- 如果价格到达那个点位，说明判断错误，必须改变立场

**成功交易的关键**：迅速而有效地适应新的市场环境——“快”（quick）这个词在英语中有两个意思：迅速、活着。反应敏捷才能在市场中生存。

### 计算机与蜡烛图

**警示**：让计算机根据蜡烛图形态自动给出交易信号，然后据此交易，而忽略整体技术面情况，可能犯下错误。

**计算机应用的局限**：
1. 蜡烛图形态需要主观解释，难以精确量化
2. 形态出现的位置比形态本身更重要，而位置判断需要整体分析
3. 非标准形态可能比标准形态更有价值，但计算机可能忽略

**实用建议**：
- 计算机可用于初步筛选形态
- 最终决策必须结合位置分析、风险收益评估和个人交易风格

## 第七部分：差异指数与日本移动平均线

### 黄金交叉与死亡交叉

**黄金交叉**：短期均线上穿长期均线，多头信号。

**死亡交叉**：短期均线下穿长期均线，空头信号。

### 差异指数（Divergence Index）

**计算公式**：
```
差异指数 = (收盘价 - 移动平均值) / 移动平均值 × 100%
```

**例如**：某日13周差异指数为-25%，表示收盘价比13周移动平均值低25%。

**三大用途**：

**① 超买超卖判定**
- 差异指数不断升高 → 市场超买
- 差异指数节节走低 → 市场超卖
- 帮助判断低价区与高价区

**② 趋势方向判定**
- 差异指数上升 → 行情上行趋势确认
- 差异指数下行 → 行情下行趋势确认

**③ 背离判定**
- 价格创出新高，但差异指数创出新低 → 空头背离
- 价格创出新低，但差异指数创出新高 → 多头背离

### 偏离指数（Deviation Index）

**计算公式**：
```
偏离指数 = 收盘价 / 移动平均值 × 100%
```

**与差异指数的关系**：
- 偏离指数103% = 差异指数+3%
- 偏离指数97% = 差异指数-3%

两条线的形态完全一样，只是纵坐标读数不同。

## 第八部分：三线反向突破图（Three-Line Break Chart）

### 概述

**定义**：不考虑时间因素的图形技术，只根据收盘价画线。只有当收盘价超越前高或前低时才添加新线。

**核心原则**：
- 收盘价超越前高 → 画白线
- 收盘价到达新低 → 画黑线
- 收盘价未创新高或新低 → 不画线

**三线规则**：如果出现三根连续白线（或黑线），必须反向突破这三根线的低点（或高点），才能画一根反转线。

### 画法详解

**初始设定**：选择基价，画第一根线

**画线判断流程**：

| 当前状态 | 条件 | 操作 |
|---------|------|------|
| 无连续三根同色线 | 收盘创出新高 | 画白线 |
| 无连续三根同色线 | 收盘创出新低 | 画黑线 |
| **有连续三根白线** | 收盘低于第三根白线低点 | 画黑转向线 |
| **有连续三根白线** | 收盘高于第三根白线高点 | 继续画白线 |
| **有连续三根黑线** | 收盘高于第三根黑线高点 | 画白转向线 |
| **有连续三根黑线** | 收盘低于第三根黑线低点 | 继续画黑线 |

### 交易技巧

**基本规则**：见白线买入，见黑线卖出。

**脖颈与黑鞋的妙用**：

**买入信号（日本谚语）**：
> “在黑鞋上面有白套装，白套装上面伸出脖颈时买入。”

- **黑鞋**：一根短黑线，表示卖压正在缓解
- **白套装**：白色转向线，突破三根连续黑线的空头反转信号
- **脖颈**：白套装后的短白线，确认买入

**卖出信号（日本谚语）**：
> “脖颈下的黑套装，黑套装下面出现黑鞋时卖出。”

- **脖颈**：上行顶端的小白线，表示买盘衰减
- **黑套装**：黑色转向线，空头反转信号
- **黑鞋**：黑套装后的小黑线，确认卖出

### 与蜡烛图的结合使用

**策略框架**：
1. 用三线反向突破图判断市场大趋势方向
2. 用蜡烛图在趋势方向内寻找入场时机

**示例**：
- 三线反向突破图出现三根连续白线 → 确认多头趋势
- 在此背景下，蜡烛图出现多头信号 → 买入时机
- 蜡烛图出现空头信号 → 作为空头仓位的平仓时机

### 调整敏感度

| 反向突破标准 | 适用对象 | 特点 |
|-------------|---------|------|
| 二线突破 | 短线交易者 | 信号多，波动频繁 |
| 三线突破 | 中线交易者 | 日本最流行 |
| 五线突破 | 长线交易者 | 信号较少 |
| 十线突破 | 超长线投资者 | 极少信号 |

一线到三线 ≈ 短期移动平均线；三线到五线 ≈ 中期移动平均线；十线 ≈ 长期移动平均线。

## 第九部分：砖形图（Renko Chart）

### 概述

**定义**：只反映价格变动幅度的图形技术，不考虑时间因素。

**核心原则**：
- 每块砖的大小相同（预先设定的固定值）
- 价格变动达到设定幅度才添加新砖
- 白砖代表上涨，黑砖代表下跌

**与三线反向突破图的区别**：三线反向突破图关注是否创出新高/新低；砖形图关注变动幅度是否达到预设值。

### 画法详解

**步骤**：
1. 设定变动幅度值（如5点）
2. 确定基价，画第一块砖
3. 比较当前收盘价与前一块砖的高点/低点
4. 如果上涨达到5点，画一块新白砖（砖高=5点）
5. 如果下跌达到5点，画一块新黑砖
6. 涨20点则画4块新砖，跌15点则画3块新砖

### 交易技巧

**基本规则**：
- 白砖出现 → 买入信号
- 黑砖出现 → 卖出信号

**优势**：能抓住行情的主升段/主跌段

**劣势**：在横盘整理时会频繁进出（砖形图是趋势跟随工具）

**最佳应用场景**：趋势明确的市场

**砖形图的价值**：
- 过滤次要价格波动
- 凸显重要趋势
- 提供长线视角

## 第十部分：钥匙图（Kagi Chart）

### 概述

**定义**：以收盘价为依据，不考虑时间因素，根据价格变动方向和幅度画线的图形技术。

**名称由来**：图中的线形像日本古代老式钥匙。

**核心特征**：
- 线有粗细之分（阳线=粗，阴线=细）
- 价格突破前高 → 线由阴转阳
- 价格突破前低 → 线由阳转阴

### 画法详解

**基本参数**：
- **转向值**：预先设定的反转幅度（可设定为绝对值或百分比）
- **百分比钥匙图**：转向值随价格自动调整（更科学）

**画线规则**：

| 条件 | 操作 |
|------|------|
| 价格沿原方向运行 | 钥匙线继续延伸 |
| 价格反向变动达到转向值 | 右移一格画反向线 |
| 价格突破前高 | 线由细变粗（阳转阴） |
| 价格突破前低 | 线由粗变细（阳转阴） |

**拐点线**：两根相反方向线之间的短横线。

### 交易技巧

**基本规则**：
- 阳线（粗线）出现 → 买入信号
- 阴线（细线）出现 → 卖出信号

**肩与腰的概念**：
- **肩**：前高点
- **腰**：前低点

**趋势判断**：
- 肩部与腰部节节抬高 → 多头强势
- 肩部与腰部逐级走低 → 空头强势

**多级突破**：
- **一级突破**：突破一个前高/前低
- **二级突破**：突破两个前高/前低
- **三级突破**：突破三个前高/前低

等待更多确认意味着更低的失误率，但也意味着更少的利润。

### 钥匙线中点的重要性

**核心原则**：修正发生在前钥匙线内部时，关注是否触及中点。

- 上涨后回抽**未触及**前长钥匙线中点 → 看多（中多方成功阻止空方）
- 上涨后回抽**触及**前长钥匙线中点 → 看空（多头力量有限）
- 下跌后反弹**未触及**前长钥匙线中点 → 看空
- 下跌后反弹**触及**前长钥匙线中点 → 看多（空头力量有限）

### 双窗口形态

**双窗底**：
- 左腰（W1）高于后面的肩部（S1）
- 右腰（W2）也高于肩部（S1）

**双窗顶**：
- 左肩（S1）低于后面的腰（W1）
- 右肩（S2）也低于腰（W1）

### 三佛形态（头肩形态）

**三佛顶** = 头肩顶

**三佛底** = 头肩底

**应用**：
- 三佛顶右肩被下穿 → 卖出信号
- 三佛底右肩被上穿 → 买入信号

**辅助判断**：
- 右佛反弹受挫于前长钥匙线中点以下 → 空头信号增强
- 卖压止步于前长钥匙线中点以上 → 多头信号增强

### 阴阳线长度的意义

| 阴阳线比较 | 市场含义 |
|-----------|---------|
| 阳线长于阴线 | 多方主导 |
| 阴线长于阳线 | 空方主导 |
| 阴阳线等长 | 市场均衡（类似十字星） |

## 第十一部分：极性转换原则

**核心概念**：
> 旧的阻力位被突破后，往往成为新的支撑位；旧的支撑位被跌破后，往往成为新的阻力位。

**蜡烛图中的体现**：
- 长白实体突破阻力后，实体本身成为支撑
- 长黑实体跌破支撑后，实体本身成为阻力
- 窗口被封闭后，原趋势结束
- 三线反向突破图中的支撑被跌破后成为阻力

**极性转换原则的价值**：
- 某个阻力/支撑被试探的次数越多，突破后的极性转换越彻底
- 有助于设定合理的止损位和目标位

┌─────────────────────────────────────────────┐

**整体技术面确认**：
- 趋势线突破后，蜡烛图反转信号更可靠

## 延伸阅读

### 1. 《日本蜡烛图技术》（原版）
**作者**：史蒂夫·尼森

这是尼森的第一本蜡烛图著作，更全面地覆盖了蜡烛图形态的完整体系。阅读顺序上，建议先读本书（新解），再读原版以获得更完整的基础知识。

### 2. 《金融市场技术分析》
**作者**：约翰·墨菲（John Murphy）

西方技术分析的经典百科全书，涵盖趋势分析、摆动指标、成交量等技术工具。与蜡烛图形成完美互补，特别推荐用于学习如何设定价格目标和量化风险收益。

### 3. 《艾略特波浪理论》
**作者**：A·J·弗罗斯特、罗伯特·普莱克特

波浪理论提供了宏观的趋势框架，与蜡烛图结合使用可以更准确地判断当前行情所处的阶段，从而决定是否顺应趋势或等待反转信号。

### 4. 《期货市场计算机分析指南》
**作者**：查尔斯·勒博、大卫·卢卡斯

尼森在本书中推荐的西方技术分析参考书，深入讲解了计算机化技术分析工具，对于想将蜡烛图与程序化分析结合的读者很有价值。

### 5. 《黄金泉——三猿金钱录》
**作者**：本间宗久（传）

日本最古老的技术分析文献之一，本间宗久的交易智慧结晶。虽然原文艰深，但其中蕴含的心理分析原则对理解蜡烛图背后的哲学思想大有裨益。
