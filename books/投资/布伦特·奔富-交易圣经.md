# 交易圣经

## 核心精髓

本书揭示了一个令人警醒的真相：**90%以上的交易者最终亏损**。然而，作者布伦特·奔富认为，成功的交易者并无独门秘笈，他们只是恪守了一些被大多数失败者忽视的基本原则。本书的核心论点简洁有力：**交易成功的唯一真正秘诀是管理好损失——输得起的人才是长期赢家**。

全书围绕**六大成功交易通则**展开，强调资金管理、方法与心理因素三大支柱的协同作用，并提供了切实可行的验证工具（TEST程序），帮助交易者构建具有正期望值的交易系统。

## 第二部分：六大成功交易原则

### 原则一：准备（Preparation）

准备是在交易之前必须完成的心理建设和风险认知，包含以下要素：

**1. 困难重重法则**
市场环境恶劣，交易充满痛苦。成功的准备意味着接受这一点，而不是期待轻松的利润。

**2. 情绪导向**
- **目标设定**：不应以"成功"或"每次正确"为目标，而应以**风险资金管理**为核心目标
- **期望管理**：20%~30%的年回报率比50%以上更易实现。"自下而上"的思维方式（先问自己什么回报能满足你）比"自上而下"（我要赚大钱）更现实

**3. 输得起才会赢**
成功交易的唯一秘诀是管理好损失。当亏损达到预设边界时，必须果断离场。

**4. 风险管理**
将交易视为风险管理而非盈利工具。优秀的交易者首先是优秀的风险管理者。

**5. 交易伙伴**
找一个值得尊重的伙伴监督你的交易表现，用财务基准衡量进展，防止自欺欺人。

**6. 财务边界**
设定学习交易的风险资金上限，如果亏光就承认自己不适合交易。

### 原则二：启蒙（Enlightenment）

启蒙将帮助你从"希望成功"转变为"追求生存"，核心概念包括：

#### 1. 破产风险（Ruin Risk）

**破产风险是打开真正交易知识大门的钥匙。**

**定义**：损失超过交易者承受能力的概率。

**降低破产风险的三种工具**：
- 减少每次交易的风险资金
- 提高方法的准确率
- 提高平均收益/平均损失的比率

**关键公式**：
```
破产风险 = f(风险资金比例, 准确率)
```

**示例**：两个交易者使用相同系统（准确率56%，盈亏比1:1），初始资金10000美元：
- 冒险者（每次风险2000美元）：5次连续亏损即破产，破产概率30%
- 保守者（每次风险500美元）：20次连续亏损才破产，破产概率1%

**核心观点**：除非破产风险接近0%，否则不应交易。

#### 2. 交易圣杯

作者认为真正的"圣杯"不是某个完美系统，而是：
- **具有正期望值的方法**（无论准确率高低）
- **足够的机会**实现这个期望

#### 3. 期望（Expectancy）

**期望公式**：
```
期望 = (准确率 × 平均收益) - (失败率 × 平均损失)
```

**示例**：准确率30%，盈亏比3:1
```
期望 = (30% × 3) - (70% × 1) = 0.90 - 0.70 = 0.20
```
即每1美元风险资金预期赚取0.20美元。

**重要结论**：
- 低准确率+高盈亏比可以产生正期望
- **趋势交易法**（准确率约30%，盈亏比3:1）是最有效的方法之一
- 交易者应关注**期望而非准确率**

#### 4. 机会（Opportunity）

仅有正期望不够，还必须有足够的机会：
- 单个市场机会有限
- 解决方案：多市场组合交易
- 假设账户可承受，每增加1个市场，机会翻倍

#### 5. 追求简单

**麦当劳测试**：如果一个青少年无法理解你的方法，它可能太复杂了。

**交易的核心**：只需识别支撑位和压力位。买入在支撑位，卖出在压力位。

#### 6. TEST验证程序

**TEST = Thirty Emailed Simulated Trades（30次电邮模拟交易）**

这是验证方法有效性的唯一可靠途径：
1. 在市场开盘前，将完整交易指令通过邮件发送给交易伙伴
2. 执行30次这样的模拟交易
3. 基于结果计算期望值
4. 如果期望为正且不依赖个别交易，则方法通过验证

**纸上模拟交易无效**：因为无法模拟真实市场的随机性和心理压力。

### 原则三：交易风格（Trading Style）

交易风格由两个维度决定：**交易方式**和**交易时限**。

#### 交易方式

| 类型 | 特点 | 准确率 | 盈亏比 |
|------|------|--------|--------|
| **趋势交易** | 顺应市场方向，等待回调或突破入场 | 25%~35% | ≥3:1 |
| **波段交易（逆势）** | 在支撑/压力位反向操作 | 50%~60% | 1:1~2:1 |

**作者偏好**：回调趋势交易——因为止损较小，虽然可能错过大趋势。

#### 交易时限

| 时限 | 持仓时间 | 适用人群 |
|------|----------|----------|
| 日交易 | 当日平仓 | 资金有限者 |
| 短期 | 1~5天 | 个人交易者 |
| 中期 | 数周 | 有一定资金规模者 |
| 长期 | 数月以上 | 机构/大账户 |

#### 选择原则

**关键因素**：财务保证、期望验证、机会数量

- 长期趋势交易需要20~30个市场组合，财务要求高
- 小额账户更适合短期波段交易
- **最终决定因素是期望和机会，而非个性偏好**

---

### 原则四：选择市场（Market Selection）

优秀市场应具备的特性：

**操作风险管理特性**：
- 价格和交易量透明度
- 高流动性（快速平仓能力）
- 24小时营业
- 零交易对方风险
- 诚信和效率
- 低交易成本

**交易特性**：
- 足够的价格波动
- 充足的历史数据用于研究
- 便于监控
- 可无条件卖空
- 可专业化
- 提供足够机会
- 增长性（可增仓）
- 可用杠杆

**作者偏好**：指数期货和外汇期货，因为它们符合所有特性。

### 原则五：三大支柱（Three Pillars）

资金管理、方法、心理因素是成功交易的三大支柱。

**重要排序**（作者观点）：
1. **资金管理**（最重要）
2. **方法**
3. **心理因素**

## 第三部分：资金管理深度解析

作者认为资金管理是"开启投机王国的金钥匙"，值得用整章篇幅详细阐述。

### 资金管理的两个目标

1. **生存**：避免破产风险
2. **巨大利润**：产生复利增长

**核心原则**：亏损时减少交易规模，盈利时增加交易规模。

### 两种资金管理模式

| 模式 | 规则 | 结果 |
|------|------|------|
| **马丁格尔** | 亏损时翻倍加仓 | 灾难——加速破产 |
| **反马丁格尔** | 亏损时减仓，盈利时加仓 | 正确——生存并增长 |

### 七种反马丁格尔策略

#### 1. 固定风险（Fixed Risk）
每笔交易风险固定金额（账户余额/单位数）

**特点**：
- 无法在亏损时减少交易
- 优点：控制单笔交易风险，适合小额账户

#### 2. 固定资本（Fixed Capital）
当账户余额达到固定资本的整数倍时增加合约数

**特点**：
- 增长最快，能产生天文数字的利润
- 缺点：灾难性损失时风险极高

#### 3. 固定比例（Fixed Ratio）
每增加一个合约需要额外实现一个"增量"利润

**特点**：
- 要求所有合约贡献相同利润
- **处理灾难性损失能力最强**
- 作者推荐策略之一

#### 4. 固定单位（Fixed Unit）
每次交易风险为账户余额/固定单位数，随余额增长而增加

**特点**：
- 增长快，但亏损时不能减少风险

#### 5. 威廉斯固定风险（Williams Fixed Risk）
```公式
合约数 = (账户余额 × 风险比例) / 最大损失
```

**特点**：
- 账户余额越大，可交易合约越多
- 作者偏爱策略之一

#### 6. 固定百分比（Fixed Percentage）
每笔交易风险占账户余额的固定百分比

**特点**：
- **专业交易者最常用的策略**
- 破产风险最低（连续亏损529次才破产，假设风险1%）
- 灵活性高

#### 7. 固定波幅（Fixed Volatility）
根据市场波动调整仓位

**特点**：
- 市场波动大时自动减仓
- 市场平静时自动加仓
- 符合"让市场告诉你仓位大小"的理念

### 策略选择建议

| 账户规模 | 推荐策略 | 理由 |
|----------|----------|------|
| 小额 | 固定比例/固定单位 | 低门槛，可起步 |
| 中等 | 固定百分比 | 平衡风险与回报 |
| 大额 | 威廉斯固定风险/固定波幅 | 专业级风控 |

### 系统终止点（System Stop）

除了交易止损，还应设置**方法的止损点**：

**目的**：
1. 确定方法层面的风险资金限额
2. 监测权益动量何时消失
3. 监测何时恢复并重新开始

**实现方式**：基于单一合约权益曲线，设置利润通道或移动止损。

## 第四部分：交易方法

### 方法的构成

```
方法 = 交易方案 + 交易计划
```

**交易方案**：识别潜在支撑位和压力位

**交易计划**：明确何时入场、何时止损、何时退出

### 交易者类型

| 类型 | 特点 |
|------|------|
| **自主交易者** | 灵活决策，但情感负担重 |
| **机械交易者** | 严格按规则执行，情感负担轻 |
| **自主机械交易者** | 两者结合，进阶形态 |

**作者建议**：新手从机械交易开始，获得稳定后再考虑自主元素。

### 技术分析学派的批判性审视

| 类型 | 代表 | 评价 |
|------|------|------|
| **预言者** | 艾略特波浪、江恩、占星术 | 主观性强，预测陷阱 |
| **梦想家** | 各种技术指标 | 滞后、二手、灵活过度 |
| **实用主义者** | 价格行为、图表模式 | 关注核心，更可靠 |

### 客观工具 vs 主观工具

**作者核心观点**：主观工具（需要输入参数的指标）是交易失败的重要原因。

**问题**：
- 不同参数值产生不同结论
- 工具反映的是交易者的主观输入，而非市场的客观事实
- 容易导致曲线拟合

**解决方案**：使用客观、独立、不需要主观输入的工具。

**示例**：
- 波动图（基于价格高低点的趋势判断）
- 固定规则（如海龟交易法则的20日/55日突破）

### 海龟交易法则示例

```
入场信号：20日高点突破做多，20日低点突破做空
止损：2 ATR
退出：10日反向突破
```

**优势**：
- 100%客观，无需主观判断
- 40年历史证明有效
- 可验证

### 斐波纳契的验证

作者进行了大规模研究（36,411次波动），结论：

> 斐波纳契比率（38.2%、61.8%等）在统计上并不比随机比例更重要。"神奇"的比率出现频率不到1%。

**启示**：需要亲自验证任何声称有效的工具或理论。

### 趋势交易的现实

**关键数据**：
- 市场只有15%的时间有明显趋势
- 趋势交易准确率约30%
- **67%的交易将亏损**

**成功趋势交易的公式**：
```
准确率 ≥ 33%，盈亏比 ≥ 3:1
→ 期望值 = 32%（每1美元赚0.32美元）
```

**警告**：不要因为连续亏损而放弃经过验证的趋势系统。

## 第五部分：心理因素

### 作者的独特视角

不同于主流观点认为"心理因素最重要"，作者认为：

> **资金管理 > 方法 > 心理因素**

但心理因素确实存在，因为它连接资金管理和方法。

### 心理因素包括

#### 1. 期望管理
不要希望每笔交易都正确，接受失败是过程的一部分。

#### 2. 贪婪管理
- 建立适度的期望目标（20%~30%）
- 期望越高，压力越大，越容易破产

#### 3. 恐惧管理
**作者的方法**：
- **短期负预期**：交易前预估可能的最大亏损
- **中期负预期**：预期会遇到严重的账户缩水期

**原理**：直面恐惧，恐惧就会减弱。

#### 4. 痛苦管理

作者认为这是最被忽视的心理因素：

> "交易是一个痛苦之地。"

**痛苦来源**：
- 亏损的痛苦
- 克制提前获利了结的冲动
- 持仓看着利润消失
- 错失大机会
- 长期账户缩水的煎熬

**应对之道**：
- 接受痛苦是交易的一部分
- 小额交易减少单笔痛苦的影响
- 机械执行减少情感消耗
- 分散注意力（作者用写作）

### 重重困难法则

> "市场必定让多数交易者失望，你将经历彻骨的痛苦。"

作者引用柯蒂斯·费斯在《海龟交易法则》中的话：

> "如果想成为成功的交易者，必须克服自负的毛病，配上谦逊的品格。"

## 第六部分：交易大师的忠告汇总

作者采访15位世界顶级交易者，核心忠告如下：

| 大师 | 忠告要点 |
|------|----------|
| **拉蒙·巴罗斯** | 精深练习——像飞行员训练一样系统化学习交易 |
| **马克·库克** | 保护利润比赚钱更难；必须接受失败并恢复 |
| **迈克尔·库克** | **缩小交易规模**，越小越好 |
| **凯文·戴维** | 简单最好——奥卡姆剃刀原则 |
| **汤姆·德马克** | 逆势交易——在疲软和下跌点操作更有效 |
| **李·格特斯** | 对每笔交易持悲观态度，关注风险而非收益 |
| **戴若·顾比** | **谦逊**——让市场告诉你该怎么做 |
| **杰夫·摩根** | 防守制胜——最好的输家才能成为最好的赢家 |
| **格雷格·莫里斯** | 规则！自律、自律、再自律 |
| **尼克·瑞吉** | **坚持不懈**——活着才能活得更好 |
| **布莱恩·谢德** | 尽早用软件回测你的交易理念 |
| **安德里亚·昂格尔** | 找到与个性兼容的方法，然后坚持执行 |
| **拉里·威廉斯** | 控制比系统更重要；没有简单答案，只有问题 |
| **王达** | 低风险交易安排是成功的基础 |

### 共同主题

1. **风险控制优先于盈利追求**
2. **简单胜于复杂**
3. **纪律和坚持是关键**
4. **谦逊和接受失败**

## 实践步骤总结

### 成为10%赢家的路径

1. **准备阶段**
   - 接受市场困难重重的现实
   - 设定适度的期望和目标
   - 寻找交易伙伴
   - 确定财务边界

2. **启蒙阶段**
   - 计算你的破产风险
   - 理解期望的概念
   - 设计具有正期望的方法
   - 用TEST验证方法

3. **选择阶段**
   - 确定交易风格（趋势/波段）
   - 选择合适的市场
   - 设定交易时限

4. **执行阶段**
   - 应用资金管理策略
   - 设计客观、简单的方法
   - 管理心理因素
   - 执行交易计划

5. **持续阶段**
   - 监测系统终止点
   - 记录和反思交易
   - 不断优化

### 作者的每日交易流程

```
1. 检查是否有交易方案
2. 确认财务边界和系统止损点
3. 计算仓位规模
4. 预估潜在亏损（计入借方）
5. 发出指令给经纪人
6. 等待执行结果
7. 更新权益曲线
8. 准备月度报告给交易伙伴
```

## 关键要点速览

1. **破产风险是核心**：除非接近0%，否则不应交易

2. **期望为王**：关注长期期望而非单笔准确率

3. **TEST验证**：30次真实模拟交易才能确认方法有效

4. **资金管理第一**：七种策略中选择适合自己的

5. **简单客观**：避免主观工具的陷阱

6. **输得起才能赢**：果断止损，放长线钓大鱼

7. **痛苦不可避免**：做好心理准备，管理痛苦

8. **坚持不懈**：使用经过验证的方法，不要轻易放弃

## 延伸阅读

### 1. 《短线交易秘诀》——拉里·威廉斯
**推荐理由**：威廉斯是资金管理领域的先驱，本书第13章"资金管理：开启投机王国的金钥匙"直接启发了布伦特。本书展示了如何用1万美元在一年内赚到110万美元（1987年），同时详细解释了凯利公式和改进方法。

### 2. 《通向财务自由之路》——范·撒普
**推荐理由**：系统交易领域的经典之作，强调了交易心理学和资金管理的重要性。与本书观点高度一致，但提供了更深入的心理学框架。

### 3. 《以交易为生》——亚历山大·埃尔德
**推荐理由**：埃尔德博士提出了"三屏"交易系统，对交易心理有独到见解。与布伦特的系统交易方法形成互补，特别适合希望理解心理因素重要性的交易者。

### 4. 《海龟交易法则》——柯蒂斯·费斯
**推荐理由**：完整披露了著名的海龟交易实验，展示了趋势跟踪系统的实际运作。作为本书中提及的方法论源头，是理解机械趋势交易的必读之作。

### 5. 《金融怪杰》——杰克·施瓦格
**推荐理由**：通过对十几位顶级交易员的深度访谈，展示了成功交易者的多元路径。与本书第12章"大师卓见"相呼应，帮助读者理解不同风格都能通向成功。

### 6. 《幽灵的礼物》——阿瑟·L·辛普森
**推荐理由**：布伦特在书中多次引用这部作品。"输得起的人才是长期赢家"这一核心观点即源于此。简短精炼，是交易哲学的入门必读。

### 7. 《专业投机原理》——维克托·斯波朗迪
**推荐理由**：系统阐述了交易心理学、风险管理和交易方法的三位一体。提供了"123法则"和"2B法则"等实用工具，与本书的方法论形成良好补充。

## 结语

布伦特·奔富通过本书传递的核心信息振聋发聩：

> **"交易的实质就是如此——输得起的人就是长期赢家。"**

这不是一本教你如何快速致富的书，而是一本帮助你建立正确交易认知、系统化交易方法、学会管理风险和接受痛苦的实战指南。作者以30年的交易经验和与15位世界顶级交易者的对话，为读者提供了一张通往10%赢家的路线图。

记住：
- **不要急**——市场永远都在，机会永远都有
- **不要贪**——适度的期望带来持续的盈利
- **不要懒**——亲自验证每一个方法，不要轻信任何人
- **不要怕**——痛苦是成长的代价，输得起才能赢

**祝你在市场中找到属于自己的生存之道。**
