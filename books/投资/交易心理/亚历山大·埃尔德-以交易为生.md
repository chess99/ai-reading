# 以交易为生（原书第2版）— 知识提炼

> 本书是交易领域的经典著作，覆盖三大支柱：**心理学、技术分析、资金管理**，加上第四要素**交易记录**，构成完整的交易框架。

---

## 一、全书核心框架

成功交易的四大支柱：
1. **心理学** — 管理情绪，避免自我毁灭
2. **技术分析** — 分析市场趋势与反转
3. **资金管理** — 控制风险，长期生存
4. **交易记录** — 从经验中学习，持续改进

## 二、个体心理学

### 2.1 交易的本质

- 交易是**负和游戏**（非零和）：赢家赚到的少于输家亏损的，因为行业（佣金+滑点）从中抽取资金
- 佣金和滑点是"无声杀手"：一个活跃交易者每年在佣金上可能花掉账户的20%
- **滑点是佣金的3倍以上**：始终用限价单，只有止损时用市价单

### 2.2 交易者的幻觉（必须破除）

| 幻觉 | 现实 |
|------|------|
| 我需要更多本金才能成功 | 失败源于认知不成熟，不是资金不足 |
| 自动交易系统能帮我赚钱 | 市场不断变化，任何系统都会失效 |
| 跟随市场大师就能赚钱 | 大师的声誉有周期，最终都会失效 |
| 我需要找到交易秘诀 | 交易对智力要求不高，关键是纪律 |

### 2.3 自我毁灭模式

- 大多数失败源于**潜意识的自毁倾向**，而非运气差或能力不足
- 症状：总把失败归咎于他人、坏运气；重复同样的错误
- 解药：**写交易日志**——记录每次建仓和清仓的理由，寻找重复模式

### 2.4 从匿名戒酒会（AA）学到的原则

埃尔德将交易失败类比于酗酒，提炼出关键原则：

**"失败者互戒会"的做法：**
- 每天早上交易前大声说："早上好，我叫XXX，我是个失败者。我有让账户亏损的倾向。"
- 这不是自我贬低，而是保持清醒的心态，避免过度自信
- **一次一天**：每天的目标是不亏损，积累成周、月、年

**核心洞见：**
- 失败者对交易的冲动 = 酒鬼对酒精的冲动
- 连续亏损是"系统失效"的信号，需要暂停审视
- 承认失去控制是康复的第一步

### 2.5 胜利者与失败者的区别

**胜利者的特征：**
- 冷静，不因单笔交易大喜大悲
- 关注长期计划，不追求刺激
- 快速从失败交易中撤出

**失败者的特征：**
- 情绪化，像赌徒一样追求兴奋感
- 无法停止交易，即使连续亏损
- 不做交易记录，逃避现实

**核心原则：**
> "市场不知道你的存在，它不会故意关照你的财富，也不会特意伤害你的资金。你要做的，只是控制自己的行为。"

### 2.6 交易者的行为准则（7条）

1. 坚定长期交易的意念（至少20年）
2. 像海绵一样学习，对任何事保持有益的怀疑
3. 不贪婪，不急于交易——市场永远在这里
4. 培养分析市场的方法：多维度确认
5. 建立资金管理计划：**生存 > 稳定增长 > 高利润**
6. 认识到交易者是系统中最薄弱的环节
7. 胜利者与失败者在思考、感受、行动上完全不同

## 三、群体心理学

### 3.1 市场是什么

- 市场 = 一大群人的集合，每个人都试图从他人口袋里赚钱
- 每个价格 = 所有市场参与者在某时刻对价值达成的**暂时性共识**
- 技术分析 = 有电脑装备的应用型社会心理学

### 3.2 群体心理特征

- 加入群体后，人变得更冲动、更轻信、更容易情绪化
- 群体比个体更简单、更原始，但力量更强大
- **不要逆趋势**：千万不要仅因为价格看起来"过高"就卖空

### 3.3 价格驱动因素

- 价格上涨/下跌的原因**不是**买卖数量的差异
- 而是**贪婪和恐惧的强度变化**
- 恐惧比贪婪强烈3倍（诺贝尔奖得主卡尼曼研究证实）
- 因此：下跌趋势中的成交量骤增比上涨中更可能意味着反转

### 3.4 趋势心理

- 上升趋势：多方贪婪推高价格，空方恐惧回补加速上涨
- 下降趋势：多方恐惧卖出加速下跌，空方贪婪维持趋势
- **顶背离**：价格创新高但指标不创新高 → 上升趋势结束信号
- **底背离**：价格创新低但指标不创新低 → 下降趋势结束信号

### 3.5 独立性的重要性

> "唯有坚持长期以独立的个体角度进行思考和操作，你才能成为一个成功的交易者。"

**实践方法：**
- 在交易前写下入场和出场条件（包括资金管理规则）
- 持仓期间**绝对不修改**计划
- 像奥德赛把自己绑在桅杆上一样，把自己绑在交易计划上

## 四、经典图表分析

### 4.1 图表的基本原则

- 开盘价：由**业余者**决定（前一晚下指令）
- 收盘价：由**专业者**主导（全天交易后的结果）
- 当收盘价高于开盘价 → 专业者比业余者更看多 → **与专业者同向**

### 4.2 柱状线解读

| 要素 | 含义 |
|------|------|
| 最高点 | 当日多方最强力量 |
| 最低点 | 当日空方最强力量 |
| 收盘价 | 多空博弈的最终结果 |
| 柱状线长度 | 多空冲突激烈程度 |

**实用规则：**
- 市场冷清时（短柱状线）进场，滑点更小
- 遇到很长柱状线时，是兑现利润的好时机
- 不要在市场拉升时追涨建仓

### 4.3 支撑位和阻力位

**支撑/阻力的强度取决于三个因素：**
1. **长度**（时间）：持续越久越强
2. **高度**（幅度）：密集区越高越强
3. **成交量**：成交量越大越强

**关键规则：**
- 支撑位和阻力位会互相转换
- 在整数价格下方几美分设置止损（避开密集订单区）
- 真正的突破伴随**大成交量**；假突破成交量很少

### 4.4 袋鼠尾（Kangaroo Tail）

**定义：** 由单根长线柱（高度是正常柱的2倍以上）及其两边普通线柱组成

- **向上袋鼠尾**：卖出信号（多方推高失败）
- **向下袋鼠尾**：买入信号（空方打压失败）
- 止损位设在袋鼠尾的**中间**位置（不是末端）
- 这是短期策略，信号在日线图上几天后消失

## 五、计算机化技术分析

### 5.1 指标分类

| 类型 | 代表指标 | 适用场景 |
|------|---------|---------|
| 趋势跟随指标 | EMA、MACD线、趋向系统 | 趋势行情 |
| 震荡指标 | MACD柱、随机指标、RSI | 震荡区间 |
| 复合指标 | 新高-新低指数、腾落指数 | 市场整体判断 |

**关键原则：** 不要用10个、20个指标，它们都基于相同的5个数据（开、高、低、收、量）

### 5.2 指数移动平均（EMA）

**参数选择：**
- 常用：22日（一个月交易日）或26日（半年交易周数）
- 两条EMA参数比例约为**2:1**（如26日和13日）
- **价值区间**：两条EMA之间的空间

**交易规则：**
- EMA上涨 → 在价格回落到EMA附近时**买入**
- EMA下跌 → 在价格反弹到EMA附近时**卖空**
- EMA走平 → 不使用趋势跟随方法

### 5.3 MACD线与MACD柱状线

**MACD线参数（标准）：** 12日EMA - 26日EMA，信号线为9日EMA

**MACD线交易规则：**
- 快线上穿慢线 → 买入信号
- 快线下穿慢线 → 卖出信号

**MACD柱状线（更重要）：**
- MACD柱 = MACD线 - 信号线
- 柱状线斜率方向比正负值更重要
- **最佳买入信号**：柱状线低于0值但斜率向上（空头力量耗尽）
- **最佳卖出信号**：柱状线高于0值但斜率向下（多头力量耗尽）

**背离信号（最强烈）：**

| 背离类型 | 条件 | 信号 |
|---------|------|------|
| 牛市背离 | 价格创新低，MACD柱底部更高 | 强烈买入 |
| 熊市背离 | 价格创新高，MACD柱顶部更低 | 强烈卖出 |

**重要条件：** 两个顶部/底部之间，MACD柱必须穿越0值线，否则不是有效背离

**最优背离时间窗口（克里·洛沃恩研究）：**
- 两个顶部/底部相隔**20-40根柱状线**（越接近20越好）
- 第二个顶部/底部长度不超过第一个的一半

### 5.4 趋向系统（DMI/ADX）

**核心指标：**
- +DI13：正向趋势线（多方力量）
- -DI13：负向趋势线（空方力量）
- ADX：平均趋向指标（趋势强度）

**交易规则：**
- +DI13 > -DI13 → 做多；-DI13 > +DI13 → 做空
- ADX上升 → 趋势增强，趋势交易有利
- ADX下降 → 趋势减弱，避免趋势跟随
- ADX跌到两条趋向线**以下**后回升4个点 → **新趋势开始**的最强信号
- ADX从高位掉头向下 → 兑现趋势利润

### 5.5 随机指标（KDJ）

**参数：** 标准5日，慢速随机（平滑一次）

**交易规则（结合周线趋势）：**
- 周线上升趋势中，等日线随机指标降到**下参考线以下**时买入
- 周线下降趋势中，等日线随机指标升到**上参考线以上**时卖空
- 超买时不买，超卖时不卖（过滤掉大部分错误交易）
- 窄浅底部 → 空方弱，上涨可能强
- 宽深底部 → 空方强，上涨可能弱

### 5.6 相对强弱指标（RSI）

**参数：** 常用7-13日

**参考线设置（5%法则）：**
- 每条线保证RSI在过去4-6个月中，超出该线的时间少于5%
- 每3个月重新调整一次

**信号优先级（从高到低）：**
1. 背离（最强）
2. 图形模式（趋势线突破）
3. 超买/超卖水平

### 5.7 强力指数指标（Force Index）

**公式：** 强力指数 = 今日成交量 × (今日收盘价 - 昨日收盘价)

**两种EMA应用：**

**2日EMA（短期，精确入场）：**
- 上升趋势中，2日EMA变**负**时买入（最佳买点）
- 下降趋势中，2日EMA变**正**时卖空（最佳卖点）
- 2日EMA比正常深5倍以上时，预期价格将很快回升

**13日EMA（中期，确认趋势）：**
- 上穿0值线 → 多头占优，买入
- 下穿0值线 → 空头占优，卖出
- 与价格背离 → 重要趋势转折点

## 六、成交量与时间

### 6.1 成交量分析原则

- 高成交量确认趋势；低成交量意味着趋势将反转
- **成交量骤增**：
  - 下跌中骤增 → 恐惧爆发，很快反转
  - 上涨中骤增 → 贪婪驱动，趋势可能继续（上涨需要更多燃料）
- 经验法则：成交量高于/低于过去两周均值25%以上，才算"高/低"

### 6.2 能量潮指标（OBV）

- 收盘价高于前日 → 成交量加入OBV
- 收盘价低于前日 → 成交量从OBV减去
- OBV创新高 → 确认上升趋势；OBV创新低 → 确认下降趋势
- **OBV与价格背离** → 强烈的反转信号

### 6.3 集散指标（A/D）

- 追踪开盘价（业余者）与收盘价（专业者）之间的关系
- 收盘价 > 开盘价 → A/D上升（专业者看多）
- **与专业者同向，与业余者反向**

### 6.4 时间框架与"5"的规律

相邻时间周期之间的关系约为5倍：
- 月线 ≈ 4.5周
- 周线 = 5天
- 日线 ≈ 5-6小时交易时间
- 小时线 ≈ 10分钟图的5倍

**原则：** 分析时至少使用两个时间周期，长周期定方向，短周期找入场点

### 6.5 指标的季节性

| 季节 | MACD柱状态 | 含义 |
|------|-----------|------|
| 春 | 0值以下但向上 | 最佳买入时机（风险小，空间大） |
| 夏 | 0值以上且向上 | 持有多仓（风险大，空间小） |
| 秋 | 0值以上但向下 | 最佳卖空时机 |
| 冬 | 0值以下且向下 | 持有空仓（风险大，空间小） |

## 七、市场指标

### 7.1 新高-新低指数（NH-NL）

**公式：** NH-NL = 创年内新高股票数 - 创年内新低股票数

**埃尔德称之为"股票市场最佳领先指标"**

**关键信号：**
- NH-NL穿越0值线 → 趋势转变
- **周度NH-NL降到-4000以下再回升** → 重要底部买入信号（"尖峰"信号，几十年才错一次）
- **周度NH-NL上升到+2500以上** → 确认牛市
- NH-NL与价格背离 → 重要反转信号

**军队比喻：** 创新高的股票是领头进攻的长官，创新低的是领头撤退的长官

### 7.2 50日均线上的股票数占比

- 超过75%后回落 → 市场顶部信号
- 低于25%后回升 → 市场底部信号
- 顶部通常宽（贪婪持续久），底部通常陡（恐惧短暂）

## 八、交易系统

### 8.1 三重滤网交易系统（Triple Screen）

**核心原理：** 使用三个时间周期过滤信号，解决趋势指标与震荡指标的矛盾

**三个滤网：**

**第一滤网（战略）— 长周期（如周线）：**
- 使用趋势跟随指标（MACD柱状线斜率）
- 确定交易方向：周线MACD柱向上 → 只做多；向下 → 只做空
- 逆趋势信号一律忽略

**第二滤网（战术）— 中周期（如日线）：**
- 使用震荡指标（随机指标、强力指数）
- 在周线上升趋势中，等日线超卖时买入
- 在周线下降趋势中，等日线超买时卖空

**第三滤网（执行）— 短周期（入场）：**
- 在第二滤网给出信号后，用追踪止损单精确入场
- 在前一天最高价上方挂买单（上升趋势中）

**实用参数示例：**
- 周线图 + 日线图 + 小时图
- 或：日线图 + 小时图 + 10分钟图

### 8.2 动力系统（Impulse System）

**构成：** EMA（趋势）+ MACD柱状线（动力）

**信号规则：**
| EMA | MACD柱 | 颜色 | 含义 |
|-----|--------|------|------|
| 上升 | 上升 | 绿色 | 强烈买入 |
| 下降 | 下降 | 红色 | 强烈卖空 |
| 其他 | 其他 | 蓝色 | 中性 |

**关键限制：**
- 绿色（多头动力）时：**禁止卖空**
- 红色（空头动力）时：**禁止买入**
- 只在蓝色时才能反向操作

### 8.3 通道交易系统

**设置：** 在EMA上下各画一条通道线，包含过去100根柱状线的95%

**交易规则：**
- 在价值区间（两条EMA之间）买入，不在通道上方买入
- 在通道下方卖空，不在价值区间卖空
- 通道上方是兑现多头利润的区域
- 通道下方是兑现空头利润的区域

**ATR通道（克里·洛沃恩方法）：**
- 在EMA上下各画1倍、2倍、3倍ATR的线
- 入场点：回调到EMA附近（负1倍ATR处）
- 盈利目标：1倍ATR处卖1/3，2倍ATR处卖1/3，3倍ATR处卖剩余
- 超过3倍ATR的波动极为罕见，通常意味着将要回调

## 九、资金管理（铁三角）

### 9.1 风险控制铁三角

**三条规则：**
1. **2%原则**：单笔交易最大风险不超过账户的2%
2. **6%原则**：单月最大亏损不超过账户的6%
3. **止损纪律**：触及止损立即执行，不讨价还价

### 9.2 2%原则详解

**公式：**
```
最大持仓量 = (账户规模 × 2%) ÷ (入场价 - 止损价)
```

**示例：**
- 账户：$30,000
- 单笔最大风险：$600（2%）
- 入场价：$50，止损价：$48.50
- 每股风险：$1.50
- 最大持仓：400股（$600 ÷ $1.50）

**意义：** 即使连续亏损50笔，账户仍有资金（每次损失2%，50次后剩余约36%）

### 9.3 6%原则详解

**公式：**
```
本月已亏损 + 所有持仓头寸的风险额度 ≤ 账户月初资金的6%
```

**操作：**
- 达到6%限制时，本月剩余时间**停止开新仓**
- 如果想开新仓，必须先平掉一个持仓来释放风险额度
- 这强制你在连续亏损后暂停，避免报复性交易

**可用风险的概念：**
- 每月总可用风险 = 账户的6%
- 每次开仓前先问：我这个月还有多少可用风险额度？

### 9.4 止损设置原则

**保护性止损：**
- 多头止损设在支撑位**以下**
- 空头止损设在阻力位**以上**
- 避免在整数价格处设置（改为整数上下几美分）

**移动止损：**
- 当价格向有利方向移动时，将止损上移至盈亏平衡点
- 再进一步移动时，锁定部分利润

**止损宽度：**
- 不能太紧（落入市场噪音区间）
- 最佳止损位至少距离入场价**1倍ATR**

### 9.5 盈利目标设置

**风险收益比：**
- 最低要求：潜在收益 / 风险 ≥ 2:1
- 优质交易：3:1或更高

**分批兑现（推荐）：**
- 1倍ATR处兑现1/3
- 2倍ATR处兑现1/3
- 3倍ATR处兑现剩余

## 十、交易工具概述

### 10.1 股票

- 个人交易者的最佳起点
- 不用担心到期日
- 流动性好的大市值股票滑点小

### 10.2 期货

- 杠杆高，风险大
- 有到期日，需要注意换月
- 持仓量是重要指标（股票没有）

### 10.3 期权

- 买方：有限损失，无限盈利
- 卖方：有限盈利，无限损失
- 隐含波动率是关键变量

### 10.4 外汇

- 没有集中成交量数据
- 可用货币期货的成交量作为替代
- 全球24小时交易，时区是挑战

## 十一、交易记录系统

### 11.1 为什么记录至关重要

> "交易记录的质量是你成功最好的预测指标。"

- 帮助打破"小赚大亏"的恶性循环
- 使你成为自己的老师
- 发现成功和失败的重复模式

### 11.2 必须记录的内容

**每笔交易记录：**
- 买入/卖出日期和价格
- 滑点和佣金
- 止损点及所有调整
- 买入理由
- 目标价位
- 最大浮盈
- 触及止损时的最大浮亏
- 使用的交易系统/策略

### 11.3 交易日志格式

**交易计划三要素（开仓前必须确定）：**
1. **入场价位**
2. **盈利目标价位**
3. **止损价位**

**绩效评估：**
- A级交易：赚取价格通道高度的30%以上
- B级交易：赚取价格通道高度的20-30%
- C级交易：赚取价格通道高度的10-20%

### 11.4 "购买前和购买后"表格

- 购买前：记录为什么要买，预期目标
- 购买后：记录实际发生了什么，与预期的差距
- 2个月后回顾：此时信号变得清晰，可以学到更多

## 十二、交易类型与时间周期

### 12.1 三种交易类型

| 类型 | 持仓时间 | 优点 | 缺点 |
|------|---------|------|------|
| 长期投资 | 月-年 | 大趋势收益 | 回撤痛苦，需要极强耐心 |
| 波段交易 | 日-周 | 机会多，风险可控 | 可能错过主升浪 |
| 日内交易 | 分钟-小时 | 无隔夜风险 | 需要极强反应力，成本高 |

**建议：** 新手从波段交易开始，先成为盈利的波段交易者，再考虑日内交易

### 12.2 日内交易的现实

- 马萨诸塞州数据：只有16%的日内交易者在6个月后仍盈利
- 日内交易的价格通道高度远小于日线图
- 除非是A级交易者（赚取通道高度30%以上），否则成本会吃掉利润

## 十三、市场分析的哲学

### 13.1 管理而非预测

> "在做金钱交易时，你不需要预测未来。你需要从市场提取信息，来寻求是否空头和多头处于控制之中。"

- 医生不预测病人是否会活下来，他处理出现的问题
- 交易者不预测价格，他评估多空力量并管理头寸

### 13.2 反向指标的使用

- 当投资顾问高度一致看多时 → 准备卖出
- 当主流杂志封面画牛时 → 多头仓位获利了结
- 同一天出现3个或更多鼓吹同一品种的广告 → 用技术指标寻找卖出信号

### 13.3 "巴斯克维尔的猎犬"信号

- 当完美的信号出现，但价格没有走向预期方向
- 这是更强烈的反向信号：表明表象之下有更基础的东西在发生变化
- 例：熊市背离出现后价格继续上涨 → 反而是更强烈的买入信号

## 延伸阅读

1. **《股市趋势技术分析》（爱德华兹、迈吉）** — 经典图表分析圣经，支撑/阻力/趋势线的系统论述

2. **《日本蜡烛图技术》（史蒂夫·尼森）** — 蜡烛图形态的权威著作，与本书互补

3. **《自律的交易者》（马克·道格拉斯）** — 深入探讨交易心理，与本书心理学部分高度互补

4. **《群体的智慧》（詹姆斯·索罗维基）** — 理解群体行为与独立决策的关系

5. **《乌合之众》（古斯塔夫·勒庞）** — 群体心理学经典，理解市场群体行为的理论基础

6. **《机械交易系统》（理查德·威斯曼）** — 系统化交易方法，适合偏向机械化交易的读者

7. **《清单革命》（阿图·葛文德）** — 理解为什么交易系统需要书面化清单

8. **《思考，快与慢》（丹尼尔·卡尼曼）** — 行为经济学，解释为什么恐惧比贪婪强3倍的科学基础

9. **《赢家通吃》（威廉姆·加拉赫）** — 揭露市场大师的真实面目，保持独立判断的参考读物

10. **《股票证券交易要诀》（恩格尔）** — 埃尔德入门时读的书，适合完全的初学者了解股市基础
