# 像经济学家一样思考

## 一、经济学的本质与性格

### 1.1 经济学的双重使命

经济学是一门兼具**科学性**与**实践性**的社会科学：

- **科学维度**：揭示经济现象背后的规律，构建逻辑严密的理论体系
- **实践维度**：消除贫困、纠正不公平、稳定物价、促进经济发展

马歇尔的名言精炼概括了经济学家的理想状态：**"火热的心，冷静的脑"**——既要有对贫困与不公的道德关切，又要有客观分析的理性能力。

> 河上肇《贫乏物语》引用拉斯金名言"生命是唯一的财富"——经济的目的不是追求财富本身，而是求道的过程。

### 1.2 经济学的两大流派

| 流派 | 核心主张 | 代表 |
|------|----------|------|
| **理性主义经济学** | 人的偏好是绝对且不变的，只由人类本性决定，不受历史、制度影响 | 新古典经济学、货币主义 |
| **历史学派/制度学派** | 人的行为受制度性因素制约，历史、文化、社会条件塑造经济行为 | 马克思主义、历史学派、制度学派 |

这两大流派贯穿经济学史，相互竞争又相互融合，共同构成经济学理论的基础。

### 1.3 理论与实证的关系

经济学研究在理论与实证的相互关联中发展：
- **理论**：提供观察视角，构建逻辑体系，模型化经济主体行为
- **实证**：通过统计数据检验理论命题，修正或推翻理论前提

二战后计算机技术发展和统计数据积累推动了**计量经济模型**的兴起，但模型的过度扩张也可能脱离经济学含义，成为阻碍理论发展的因素。

## 二、古典经济学：从斯密到马克思

### 2.1 亚当·斯密与《国富论》

#### 思想背景
斯密先以道德哲学家身份著《道德情操论》（1759），再写《国富论》（1776）。这一顺序至关重要：

- **《道德情操论》**：以"同情"（sympathy）为核心，阐明人性本质上是社会性的。公民社会是有着同情能力的人的社会集体。
- **《国富论》**：是《道德情操论》的延伸，阐明新兴公民社会遵循的经济原理。

#### 核心理论框架

**劳动分工与生产力**
- 扣针工厂例子：10人分工一天可生产48,000枚，分工不是理论设计的产物，而是人们追求各自利益时自然产生的
- 分工受市场规模制约，以自由交换为前提

**自然价格**
- 在自然价格下，每个人自由行动和选择，生产要素实现有效配置
- "看不见的手"：每个人追求自身利益，社会整体却实现和谐状态
- 自然价格由工资、利润、地租三部分构成，对应工人、资本家、地主三大阶级

**对重商主义与重农主义的双重批判**
- 批判重商主义：依靠牺牲他国利益获取财富，扼杀自身经济发展
- 批判重农主义：农业保护政策导致市场价格过度偏离自然价格
- 结论：只有以自由竞争为基础的市场经济制度才能促成新产业形成，提高公民生活水准

**国家职能（《国富论》第五篇）**
1. 国防义务
2. 司法义务
3. 建立和维持公共设施和公共工程（即**社会共通资本**的早期表述）

### 2.2 李嘉图与马尔萨斯

**李嘉图的核心贡献**

| 理论 | 内容 |
|------|------|
| **劳动价值说** | 商品价值依赖于生产商品直接或间接投入的劳动量 |
| **分配理论** | 地租由最优土地与劣等土地产量之差决定；利润率长期呈下降趋势 |
| **比较成本优势** | 即使一国在所有商品上都有绝对优势，各国专注于相对优势的商品，自由贸易对双方都有利 |

**李嘉图理论的三个隐含前提**（宇泽批判性分析）：
1. 劳动与其他生产要素不能相互替代
2. 生产要素可以自由、无成本地在产业间转移
3. 所有生产要素归私人所有（忽视了社会共通资本的存在）

**马尔萨斯**：人口按等比数列增长，粮食生产只能按等差数列增长，提出总供给与总需求平衡问题。

### 2.3 卡尔·马克思

**核心贡献**

- **唯物史观**：经济基础决定政治、法律、文化等上层建筑
- **剩余价值理论**：资产阶级无偿占有劳动者创造的剩余价值（商品价值减去工资的部分）
- **历史观**：将资本主义生产方式看作历史过程的一个阶段，指出其内在矛盾必然导向社会主义

**马克思的"资本"概念**
资本以"货币资本→生产资本→商品资本→货币资本"形式循环，资本家不过是将资本意图付诸实践的工具。资本的本质是增殖，手段是最大限度剥削劳动力。

**阶级观的演进**
- 斯密：地主、资本家、工人三大阶级，在自然价格下维持和谐关系
- 李嘉图：三阶级之间存在对立，但新兴工业社会有发展机会
- 马克思：资本家与工人阶级的矛盾和对立是资本主义的**根本特征**

**关于社会主义计划经济的讨论**
- 米塞斯（1920）：合理计划经济需解数百万方程，计算量不可能实现
- 哈耶克：价格在市场经济中有不可替代的信息传递功能，计划经济必然走向集权
- 兰格-勒纳方案：竞争性社会主义，将价格机制引入计划经济
- 宇泽评价：中国社会主义与东欧不同，顺应自然、历史、文化条件制定经济体制，具有独立性

## 三、现代经济学的诞生：新古典经济学

### 3.1 边际革命（1870年代）

三本改变经济学的著作：
- 杰文斯《政治经济学理论》（1871）
- 门格尔《国民经济学原理》（1871）
- 瓦尔拉斯《纯粹经济学要义》（1874-1877）

共同转向：从**生产费用论**（劳动决定价值）到**边际效用论**（消费者主观效用决定价值）

**奥地利学派归属理论（门格尔）**
商品价值由最终消费者使用时获得的主观效用决定，各生产要素的价值由其边际产出贡献决定。

### 3.2 瓦尔拉斯一般均衡理论

**消费者行为**
- 无差异曲线：描述消费者偏好，凸向原点
- 预算约束：P₁C₁ + P₂C₂ ≤ Y
- 均衡条件：**边际替代率 = 价格比率**（P₁/P₂）

**生产者行为**
- 生产函数：Q = F(N)（短期）
- 均衡条件：**劳动边际产量 = 实际工资率**（边际原理）

**一般均衡**
所有商品需求量等于供给量时的价格体系（W, P₁,...,Pₘ）形成均衡。
瓦尔拉斯法则：全部利润分配给消费者时，独立方程数等于未知数数目。

**希克斯的"周"**
- 每周一通过"摸索"（Tâtonnement）拍卖发现均衡价格
- 周二至周五按计划生产，周六周日消费
- **罗宾逊的批判**："周"同时具有完成摸索的**长期性**和外部条件不变的**短期性**，这是一个自相矛盾的概念，揭示了一般均衡理论的基本矛盾

**费雪的时间偏好理论**
- 用"时间偏好率"（主观利率）比较现在与未来消费
- 最优消费计划：时间偏好率 = 实际利率
- 费雪方程：名义利率 = 实际利率 + 通货膨胀预期

### 3.3 新古典经济学的理论前提（六大假设）

| 前提 | 内容 |
|------|------|
| **生产资料私有制** | 所有稀缺资源归某个经济主体私有，可自由处置和交易 |
| **经济人假设** | 人基于独立的主观价值标准理性行动，与历史、文化无关 |
| **主观价值独立性** | 偏好不受他人影响，可用无差异曲线表达 |
| **生产资料可塑性** | 生产要素可无成本、无时间延迟地在不同用途间自由转换 |
| **生产时间瞬时性** | 生产时间为零或可忽略 |
| **市场均衡稳定性** | 需求供给失衡时，价格迅速调整，瞬间恢复均衡 |

### 3.4 新古典经济学的基本命题

1. **帕累托最优**：竞争均衡满足帕累托最优（提高一人效用必降低他人效用）。注意：只关注效率，不涉及公平。
2. **充分就业**：劳动供给全部被使用，充分就业是常态
3. **货币中性**：物价上涨率等于货币供给增长率，货币不影响实体经济
4. **税收中性**：不影响市场价格体系的直接税（所得税）比间接税更合理
5. **自由贸易**：废除贸易壁垒使所有国家受益
6. **阿罗不可能定律**：在民主约束下，不存在能将个人价值标准加总为社会价值标准的方法

## 四、凡勃伦：新古典经济学的制度批判者

### 4.1 核心批判

凡勃伦（1898-1899年论文）最早系统批判新古典经济学，并最早使用"新古典经济学"这一术语。他的批判直指六大前提条件的不现实性。

**"经济人"的批判**（《有闲阶级论》，1899）
> "享乐主义关于个体的概念是将人视作闪电般地计算快乐与痛苦的计算器，他像一个追求快乐的摇摆的同质的小球……"

人的经济活动很大程度上由文化、社会、历史因素，尤其是**制度**所决定。

### 4.2 《企业论》（1904）的核心洞见

**生产资料的固定性**
- 机器一旦生产并配置到工厂，几乎不可能再转做他用
- 企业是有机组织，是具有自身认同的实体，而非生产资料的简单集合
- 这与新古典经济学的"可塑性"假设截然相反

**产业与营利的背离**
- 制造者本能（追求生产效率）vs. 营利企业追求利润的经营方法
- 这种紧张关系导致生产设备运转不足和**非自愿失业是常态**
- 帕累托最优在现代资本主义制度条件下不再适用

**金融资产市场的不稳定性**
- 股票市场中投机性动机主导，市场价格与企业实际价值背离
- 投机引发更多投机 → 泡沫崩溃 → 金融恐慌 → 有效需求减少 → 非自愿失业增加 → 慢性经济停滞
- 这一分析比凯恩斯早25年，精准预测了1929年大萧条的机制

**政策处方**
凡勃伦认为：要么让人们更加浪费（扩大消费），要么**政府扩大公共投资**提高有效需求。这与凯恩斯的政策建议不谋而合，但凡勃伦晚年转向技术专家治国的计划经济方向（《工程师和价格制度》，1921）。

## 五、凯恩斯经济学

### 5.1 凯恩斯提出的问题

《通论》（1936）的核心：批判新古典经济学，证明**非自愿失业是常态，充分就业只是极端特例**。

凯恩斯在《通论》最后一章明确指出资本主义经济的两大缺陷：
1. 不能提供充分就业
2. 财富和收入分配不公平合理

### 5.2 凯恩斯的理论前提

**凡勃伦-凯恩斯企业观**（核心出发点）
- 企业是有机组织，是按照理性原则行动的实体单位
- 企业决定生产、投资、技术开发，左右资本主义经济的循环方式
- 与新古典经济学中"追求利润最大化的生产协调人"本质不同

**两部门划分**
- **企业部门**：决定生产和投资（凡勃伦式固定生产资料组织）
- **居民部门**：决定消费和储蓄

**阶级观**（《货币改革论》，1923）
三个阶级：投资家（企业化身）、劳动者（出售劳动力）、食利者（持有金融资产）

### 5.3 两个基本前提的取舍

| 前提 | 内容 | 凯恩斯态度 |
|------|------|-----------|
| 第一前提 | 劳动需求量在劳动边际产量=实际工资时确定 | **接受** |
| 第二前提 | 劳动供给量在劳动边际负效用=实际工资时确定 | **否定** |

凯恩斯否定第二前提，认为非自愿失业是常态，这就是他的著作取名《**通**论》的原因——充分就业才是极端特例。

### 5.4 投资理论

**投资的本质**：固定生产资料的积累（区别于库存投资）

**资本边际效率**：投资水平每增加一单位，企业利润从现在至将来的增量（贴现后）。当资本边际效率 = 长期（实际）利率时，投资水平最优。

**预期的作用**
- 短期预期：生产到销售期间对市场条件的预期（与现实差距小）
- 长期预期：固定资产使用寿命期间对市场条件的预期（影响投资，高度不稳定）

长期预期的不稳定性是资本主义经济周期不稳定的重要原因。

### 5.5 有效需求理论

**总供给曲线（SS）**：就业量N与国民收入Y的关系（向右上方倾斜）

**总需求曲线（DD）**：
$$X = C + I + G$$
- C（消费）：主要取决于国民收入，边际消费倾向 < 1
- I（投资）：取决于长期预期和资本边际效率
- G（政府支出）：包含自主部分和收入相关部分

**有效需求**：SS与DD曲线交点对应的需求水平，决定实际就业量。

**关键区别**：
- 新古典经济学：投资=储蓄，SS=DD（萨伊定律成立），充分就业是常态
- 凯恩斯：企业与居民是性质不同的经济主体，SS≠DD，有效需求不足导致非自愿失业

### 5.6 流动性偏好理论

市场利率由**货币需求**（L）和**货币供给**（M）相等时决定。

货币需求取决于：
- 交易动机
- 投机性目的（预防未来利率变化）
- 安全性目的

货币供给由金融管理当局控制，市场利率在LL与MM曲线交点处确定。

### 5.7 希克斯IS-LM分析

**IS曲线**：市场利率i与有效需求决定的就业量N的关系（向右下方倾斜）
- 利率上升 → 投资减少 → 总需求下降 → 就业量减少
- IS = "投资(I) = 储蓄(S)"

**LM曲线**：就业量N与货币需求等于货币供给时的利率i的关系（向右上方倾斜）
- 就业增加 → 货币需求上升 → 利率上升
- LM = "流动性(L) = 货币供给(M)"

**均衡**：IS与LM交点E，就业量N一般与充分就业水平Nf不同，差值Nf-N即**非自愿失业规模**。

**政策含义**
- 扩大有效需求：增加财政支出（IS上移）、降低所得税（IS上移）
- 间接扩大有效需求：增加货币供给（LM下移）→ 降低利率 → 刺激投资

### 5.8 凯恩斯的理性主义

凯恩斯追求基于理性的财政政策和货币政策，实现充分就业和收入分配平等化。他相信，拥有优越认知能力的精英人士（"哈维路的文化前提"）能够为政府提供理性的政策建议——这种精英主义思想后来成为批判凯恩斯主义的靶点之一。

## 六、战后经济学（1945-1970年代）

### 6.1 凯恩斯革命的扩散

**萨缪尔森的《经济学》（1948）**：第一本以IS-LM分析为基础的教科书，凯恩斯主义成为战后经济学主流。

**计量经济模型的发展**：
- 考尔斯研究所（马尔沙克、库普曼斯）奠定理论基础
- 克莱因《美国的一个经济计量模型》（1950）：6个方程组成的计量经济模型
- 后来规模不断扩大，成为政策预测的重要工具，但也逐渐脱离经济理论含义

### 6.2 麦卡锡主义与经济学

20世纪40-50年代，麦卡锡主义将凯恩斯主义与共产主义并列为危险思想：
- 斯威齐被迫离开哈佛大学
- 艾奥瓦州立大学"人造奶油论文事件"（舒尔茨等教授辞职）
- 伊利诺伊州立大学"伊利诺伊事件"（莫迪格利安尼、赫维克兹等被迫离职）

**宇泽的评价**：正是众多经济学家的牺牲和坚守，维护了学术自由，使美国经济学在50-60年代迎来黄金时代。

### 6.3 经济增长理论

**哈罗德-多马模型**
- 自然增长率（充分就业前提下的增长率）vs. 保证增长率（资本积累决定的增长率）
- 两者由完全不同因素决定，一致纯属偶然
- 结论：稳定均衡经济增长极其不稳定，如同"在刀尖上行走"

**新古典经济增长理论**（索洛、托宾、斯旺等）
- 引入资本与劳动的替代关系
- 通过资本装备率变化保证经济增长稳定性
- 缺陷：使用总量生产函数，只适用于单一商品情况

**二元经济增长模型**（米德、宇泽）
- 区分资本品和消费品两个经济部门
- 明确投资品和消费品的技术条件与收入分配、储蓄倾向之间的复杂关系

### 6.4 越南战争与经济学的转向

1962年起美国介入越南战争，引发大学生大规模反战运动。经济学专业学生比例极高，他们的觉醒直接推动了经济学的转型，为此后批判新古典经济学和凯恩斯主义的思潮埋下伏笔。

## 七、琼·罗宾逊与经济理论的第二次危机

### 7.1 琼·罗宾逊的主要贡献

| 著作/事件 | 贡献 |
|-----------|------|
| 《不完全竞争经济学》（1933） | 分析不完全竞争市场，得出市场机制资源配置非最优的结论 |
| 凯恩斯学术圈（1931起） | 与卡恩等人共同构建凯恩斯理论，对《通论》诞生有重大影响 |
| 《资本积累论》（1956） | 集大成之作，构建分析资本积累长期过程的理论框架 |
| "两个剑桥之争" | 证明新古典经济学利润率概念存在逻辑矛盾（转型问题） |
| 《经济学的异端》（1971） | 批判战后经济学的"增长神话"，指出时间因素被剥离的根本缺陷 |

### 7.2 对IS-LM分析的批判

罗宾逊从一开始就批判希克斯的IS-LM分析：
- IS-LM过于拘泥于**静态均衡分析框架**
- 大大偏离了凯恩斯学术圈的**动态不均衡**分析思想
- 凯恩斯《通论》本身也没有准确体现这种动态分析框架

### 7.3 "经济理论的第二次危机"（1971年）

**第一次危机**（1930年代）：大萧条击碎新古典经济学神话，凯恩斯《通论》提供解决方案。

**第二次危机**（1960-70年代）：
- 通货膨胀、失业、国际收支失衡螺旋上升
- 凯恩斯主义财政和货币政策完全失效
- 城乡差距扩大，环境问题爆发
- 滞胀（通胀与失业并存）出现

罗宾逊的核心诊断：经济学家忘记了"人类生活是在时间中进行的"，将时间因素从理论中剔除，实质上复活了凯恩斯以前的旧理论。

## 八、反凯恩斯主义经济学的盛行（1970年代）

### 8.1 理性预期假说（卢卡斯，1972）

**核心假设**：每个人能够正确认识未来市场价格的客观概率分布，并据此采取最优行动。

**卢卡斯模型的"两岛寓言"**：
- 两个小岛，固定的人口分配概率，共同的货币管理机构
- 居民能准确知道货币供给增长率和人口增长率
- 结论：货币数量理论得到严密证明，货币中性成立

**宇泽的根本批判**：
1. 假设人们能准确认识需求曲线和供给曲线 → 意味着对所有消费者偏好和生产者技术都有准确认识 → **否定了市场制度的分权性**（每个人都需要窥探他人隐私）
2. 在全知全能的假设下，市场制度本身就没有存在的必要性
3. 理性预期假说实际上是**从否定市场制度的前提出发，来证明市场制度的效率性**——这是一个根本性的逻辑悖论

### 8.2 弗里德曼的《实证经济学》（1953）

核心主张：评价经济理论不应考虑前提条件是否合理，只看推导出的命题能否说明现实。

**宇泽批判**：这与凯恩斯《通论》开篇的想法截然相反。凯恩斯认为，理论合理性的主要依据是**前提是否准确反映现实的制度性、社会性及技术性条件**，而非结论是否令人满意。

### 8.3 供给学派

**拉弗曲线**
- 主张：降低所得税平均税率，政府税收总额反而可能增加
- 理论基础：劳动供给随税后工资提高而增加（新古典经济学劳动供给理论）
- **根本缺陷**：完全忽视需求侧约束；假设每个劳动者能理性决定工作时间，忽视了企业决定雇用的实际机制
- 政治影响：成为里根政府减税政策的理论依据，导致美国历史上最大规模的预算赤字

**里根政府的三大纲领及其后果**：
1. 大幅增加军费开支 ✓
2. 三年内降低所得税30% ✓
3. 消除预算赤字 ✗（反而产生2000亿美元赤字）

**费尔德斯坦的主张**：社会保障制度抑制民间储蓄，导致投资不足。但其计算被发现存在重大错误，结论不可靠。

### 8.4 货币主义经济学

**思想渊源**：休谟-费雪货币数量方程 PO = MV

**弗里德曼的货币数量说新表述**（1956）：
- 借用凯恩斯-托宾的资产选择理论（未引用托宾）
- 马歇尔k值（货币需求/名义收入）为固定值或变化率极小
- 结论：物价上涨率等于货币供给增长率

**宇泽的批判**：
- 用恶性通货膨胀验证货币数量说，再推广到高度发达金融制度的现代资本主义，毫无根据
- 恶性通货膨胀时金融制度崩溃，货币数量说当然成立；但这不能说明正常情况下的适用性

**政策影响**：废除金融制度规制、社会共通资本私有化、平衡预算、关注货币供给的货币政策、自由贸易、资本自由化、浮动汇率制度

### 8.5 宇泽对反凯恩斯主义的总体评价

> 这些理论在理论完整性和现实合理性上都是粗浅的，内部还不时出现严重矛盾。充其量仅是通往新经济研究范式道路上昙花一现的现象。

莱卡赫曼《陷入困境的经济学家》（1976）指出：应回归亚当·斯密、卡尔·马克思、托斯丹·凡勃伦、约翰·梅纳德·凯恩斯四位经济学家的思想中汲取智慧——宇泽对此深表赞同。

## 九、现代经济学的新方向

### 9.1 两种不均衡

**市场不均衡**：需求与供给的背离无法通过市场机制完全消除，有时经济陷入螺旋恶化。原因：经济主体行为受文化、历史、政治等结构性因素影响，不能仅用理性经济人模型分析。

**社会不均衡**：私有资本与社会共通资本的平衡比例遭到破坏，政府政策无法完全消除市场经济内部的不稳定因素。

### 9.2 动态不均衡理论：霍特里-小谷模型

**理论背景**：凯恩斯学术圈（以理查德·卡恩为中心）提倡动态不均衡分析，但凯恩斯《通论》未能准确体现这一思想。小谷清（1978年论文）将其公式化。

**凯恩斯《通论》的理论缺陷**：
- 将固定资本投资与库存投资混为一谈
- 若库存变化被计入投资，则总需求恒等于总供给（有效需求理论失效）
- 正确做法：遵循霍特里的主张，投资仅指**固定资本积累**

**霍特里过程 vs. 凯恩斯过程**

| 比较维度 | 霍特里过程 | 凯恩斯过程 |
|----------|------------|------------|
| 货币供给增加的传导 | 短期利率下降→库存成本降低→库存需求增加→**库存市场价格上涨**→生产扩大→就业增加 | 长期利率下降→投资边际效率提高→投资增加→总需求上移 |
| 价格决定机制 | 库存市场（组织化程度高）均衡价格决定物价水平 | 未明确公式化 |
| 现实解释力 | **更强**（更接近实际经济周期机制） | 较弱 |

**霍特里-小谷模型的三个就业量**

在任意时点t：
- **Nat**（实际就业量）：由过去的经济行为决定，当期不能调节
- **Not**（最优就业量）：库存市场价格pt对应的劳动边际产量=实际工资时的就业量
- **Net**（有效就业量）：假设雇用可变时，有效需求对应的就业量

三者一般不相等。当Not < Nat时，企业减少雇用；当Net < Nat时，库存积累，库存市场价格下降，Not进一步降低……形成图12所示的循环动态过程。

**政策含义**：
- 凯恩斯-希克斯均衡（三者相等）是均衡点，但不一定稳定
- 20世纪70年代的世界实际上处于霍特里-小谷过程的循环或不稳定状态，而非IS-LM均衡
- 这解释了为何计量经济模型的政策预测在70年代频频失效

### 9.3 社会共通资本理论

**定义**
稀缺资源按所有制分为两类：
- **私有资本**：属于各经济主体，可自由处置和市场交易
- **社会共通资本**：不能被私有，完全处于社会管理之下，按社会性标准供应和分配

**分类**

| 类别 | 例子 |
|------|------|
| 自然资本 | 空气、河流、森林 |
| 社会资本（基础设施） | 堤防、道路、港湾、公园、下水道、电力、铁路 |
| 制度资本 | 司法、教育、医疗、城市、市场本身 |

**为何需要社会共通资本**
- 若维持基本生活所必需的服务通过追求私利的市场提供，会导致社会或文化层面的不良后果、经济低效或分配不公
- 公民基本权利（自由权、生活权）的保障需要社会共通资本

**社会共通资本与社会稳定性**
若仅通过收入再分配政策（而非社会共通资本制度）保障公民基本权利，会引发价格螺旋上涨：
- 补助低收入群体 → 基本生活商品需求增加 → 价格上涨 → 维持基本生活所需收入上升 → 需要更多补助 → 社会不稳定

**生产函数中的社会共通资本**
$$Q_j = F_j(K_j, N_j, X_j, X, V)$$
- V（社会共通资本存量）增加 → 生产量增加
- X（整体使用量）增加 → 生产量**降低**（混杂效应/拥挤效应）

最优配置条件：社会共通资本生产的服务的价格 = 使用该服务的**边际社会成本**（所有生产主体生产量减少的合计值）

**与亚当·斯密的联系**
斯密在《国富论》第五篇中提出国家应建立和维持公共设施和公共工程，这正是社会共通资本思想的早期表述。

## 十、新经济学的曙光

### 10.1 阿克洛夫与斯蒂格利茨的新方向

**阿克洛夫的"柠檬市场"（1971）**
- 二手汽车市场中，卖家知道质量，买家不知道
- 信息不对称导致市场失灵
- 将经济主体定义为具有历史、社会、文化属性的社会人，而非抽象经济人
- 将信息、知识、技术等制度因素与经济周期联系加以公式化
- 汲取凡勃伦制度学派精华，公式化了罗宾逊的问题意识

**斯蒂格利茨**：在产业组织、金融市场、风险、道德风险、地方公共服务等领域开展先驱性研究。

### 10.2 环境经济学与社会共通资本

地球温室化效应等环境问题：
- 当代人与子孙后代之间的公正分配问题
- 发达国家与发展中国家之间的经济差距问题
- 需要以**社会共通资本**为轴线构建新的经济学分析框架

              ↓                           ↓

**宇泽的核心主张**：
1. 经济学必须回归**人的社会性**，拒绝抽象的"经济人"假设
2. 必须正视**时间的不可逆性**——过去不可改变，未来充满不确定性
3. 社会共通资本是市场经济稳定运行的必要基础，不能被私有化
4. 效率固然重要，但**公平与公正**是经济学不能回避的核心问题
5. 好的经济学需要同时具备"火热的心"（道德关切）和"冷静的脑"（理性分析）

## 延伸阅读

### 经济学史与思想史

1. **《国富论》亚当·斯密** — 本书多次引用的经济学原点，直接阅读原典，理解斯密关于分工、自然价格、公民社会的完整思想。

2. **《就业、利息与货币通论》约翰·梅纳德·凯恩斯** — 本书的核心文本，虽晦涩难懂，但配合本书的解读框架，可以更深入理解凯恩斯的真实意图。

3. **《有闲阶级论》托斯丹·凡勃伦** — 对新古典经济学"经济人"假设最犀利的批判，理解制度经济学的入门必读。

4. **《政治经济学及赋税原理》大卫·李嘉图** — 理解比较优势、分配理论等古典经济学核心概念的第一手资料。

### 宏观经济学与经济周期

5. **《价值与资本》约翰·希克斯** — 本书详细介绍了希克斯对瓦尔拉斯一般均衡理论的系统化阐述，以及IS-LM分析的原始来源。

6. **《经济学的异端》琼·罗宾逊** — 本书大量讨论的罗宾逊的批判性著作，对战后经济学神话的深刻解构。

7. **《资本积累论》琼·罗宾逊** — 罗宾逊集大成之作，尝试构建分析资本主义积累长期过程的新框架。

### 制度经济学与现代经济学

8. **《企业论》托斯丹·凡勃伦** — 预见大萧条、分析现代资本主义病理的经典，与本书第5章直接对应。

9. **《不均衡理论》小谷清** — 本书第10章动态不均衡理论的直接来源，将凯恩斯学术圈的动态不均衡思想公式化。

10. **《近代企业和私有财产》阿道夫·伯利、加德纳·米恩斯** — 实证验证凡勃伦-凯恩斯企业观的经典，关于所有权与经营权分离的奠基之作。

### 经济政策与批判

11. **《通往奴役之路》弗里德里希·哈耶克** — 本书的重要对话对象，理解计划经济批判与市场制度辩护的必读文本，与宇泽的社会共通资本理论形成对照。

12. **《陷入困境的经济学家》罗伯特·莱卡赫曼** — 本书第9章重点介绍，对20世纪70年代反凯恩斯主义经济学的系统批判，与宇泽立场高度一致。

13. **《社会共通资本》宇泽弘文** — 本书作者的另一部重要著作，对本书第10章社会共通资本理论的全面展开，是理解宇泽思想体系的必读补充。
