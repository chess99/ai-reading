# 政治经济学及赋税原理

## 概述

大卫·李嘉图的《政治经济学及赋税原理》（1817年首版，1821年第三版）是继亚当·斯密《国富论》之后最伟大的经济学著作，标志着英国古典政治经济学的完成。本书在批判继承斯密劳动价值论的基础上，建立了以劳动价值论为基础的分配理论体系，揭示了工资、利润和地租之间的对立关系，为马克思主义政治经济学的形成奠定了重要基础。

卡尔·马克思评价本书指出，全部理论精华已包含在前六章之中，尤其是前两章，无论从独创性、见解的统一性、集中性和深刻性而言，都给予了读者理论上的满足感。

## 第二篇 分配理论

### 一、地租理论

#### 地租的严格定义

李嘉图澄清了地租概念的混乱：

> **地租是为使用土地原有的和不可摧毁的土壤生产力而付给地主的部分土地产品**

这与通俗语言中把农场主每年支付给地主的一切都称为地租不同。严格的地租仅指使用土地自然生产力所支付的报酬，不包括资本利息和利润。

#### 地租产生的原因

地租产生于两个条件：
1. **土地数量有限**
2. **土地质量不一致**

在社会初期，当土地辽阔、任人随意种植时，不会有地租。随着人口增长，必须耕种质量较差的土地时，肥沃土地便开始产生地租。

#### 级差地租理论

李嘉图建立了著名的**级差地租理论**：

```
土地等级与产量（等量资本和劳动）：
- 一等土地：180夸脱
- 二等土地：170夸脱
- 三等土地：160夸脱

地租计算：
- 一等土地地租 = 180 - 160 = 20夸脱
- 二等土地地租 = 170 - 160 = 10夸脱
- 三等土地：不支付地租（决定谷物价格）
```

**核心原理**：农产品价格由**不支付地租的土地**（边际土地）上生产所需劳动量决定。

#### 地租与价格的关系

> **谷物的价值受在那等土地上进行生产时所投入的劳动量所决定。谷物的价格昂贵不是因为支付了地租，但支付地租却是因为谷物价格昂贵。**

李嘉图的重要论断：
- 如果地主放弃全部地租，也不会使谷物降价
- 这种方法只会使农场主生活如绅士一般，却不会减少在生产力最低土地上种植农产品所需的劳动量
- **地租不是商品价格的组成部分**

#### 影响地租的因素

| 因素 | 对地租的影响 |
|------|-------------|
| 人口增加 | 地租上涨 |
| 农业改良（提高土地生产力） | 初期降低地租，最终因人口增加而恢复或提高 |
| 农业改良（减少劳动投入） | 降低地租 |
| 资本减少 | 地租下降 |

### 二、工资理论

#### 工资的自然价格与市场价格

- **自然价格**：让劳动者继续生存并维持其后代所必需的价格，取决于维持劳动者及其家庭所必需的食品、生活必需品和便利设施的价格
- **市场价格**：按供求比例实际支付的价格

#### 工资变动趋势

随着社会进步，劳动的自然价格有**上涨倾向**，原因：
- 农产品生产愈发困难
- 食品和必需品的价格上涨

但农业改良和进口食品市场的新发现可能暂时阻挡这一趋势。

#### 工资与人口的关系

```
高工资 → 人口增长 → 劳动者数量增加 → 工资下降至自然价格水平
```

#### 工资变动的实际影响

**重要区分**：货币工资上涨 ≠ 实际工资上涨

示例分析：
```
初始状态：
- 谷物价格：4英镑/夸脱
- 年工资：24英镑 = 6夸脱谷物价值
- 工资的一半用于购买谷物，另一半用于其他商品

当谷物涨至5英镑/夸脱：
- 如果工资涨至25英镑（涨幅约4%）
- 25英镑 = 5夸脱谷物
- 劳动者得到更多货币工资，但谷物工资减少
```

李嘉图的结论：
> **劳动者的状况一般会下降，而地主的状况则总会得到改善。**

### 三、利润理论

#### 利润的决定因素

李嘉图的核心命题：
> **利润取决于工资的高低，工资取决于必需品的价格，必需品的价格主要依赖于食物价格**

#### 利润与分配的数学分析

沿用地租分析中的例子：

```
初始状态（一夸脱4英镑）：
- 产量180夸脱 → 总价值720英镑
- 支付地租：0（最差土地）
- 支付工资：480英镑（10人×48英镑）
- 农场主利润：240英镑
- 利润率：约8%

产量减少至170夸脱（需耕种较差土地）：
- 价格必须上涨至4英镑4先令8便士
- 总价值仍为720英镑
- 地租：10夸脱 = 42.7英镑
- 工资：必须增加（因为必需品涨价）
- 农场主利润：下降
```

**关键发现**：无论小麦售价为4英镑、4英镑10先令或5英镑2先令10便士，农场主支付地租后从剩余中所得到的实际价值都是相同的。

#### 利润下降的趋势

> **利润的自然趋势是下降的，因为在社会进步和财富增长中，必需的食品增加量是通过牺牲越来越多的劳动而获得的。**

**极限情况**：当每夸脱谷物售价为20英镑时，全部纯收入属于地主，因为原先生产180夸脱的等量劳动现在仅能生产36夸脱。

#### 利润率与资本积累

| 资本增加 | 利润率 | 利润总额 |
|---------|--------|---------|
| 10万英镑 | 20% | 2万英镑 |
| 20万英镑 | 19% | 3.8万英镑 |
| 30万英镑 | 18% | 5.4万英镑 |
| 100万英镑 | 7% | 7万英镑 |
| 110万英镑 | 6% | 6.6万英镑 |

**结论**：当资本积累到很大数额并且利润下降后，进一步积累就会减少利润总额。

## 第三篇 对外贸易理论

### 一、比较优势原理

#### 核心命题

> **各国都必然将其资本和劳动用于对本国最有利的方面**

李嘉图的著名例子：

```
英国生产布匹：100人/年劳动
英国生产葡萄酒：120人/年劳动

葡萄牙生产葡萄酒：80人/年劳动
葡萄牙生产布匹：90人/年劳动

结论：
- 英国应专门生产布匹，出口换取葡萄酒
- 葡萄牙应专门生产葡萄酒，出口换取布匹
- 即使葡萄牙能以90人劳动生产布匹，仍从需100人劳动的英国进口
- 因为葡萄牙宁愿把资本投在葡萄酒生产上
```

#### 与斯密的区别

斯密认为国际贸易基于**绝对优势**——各国生产各自具有绝对成本优势的商品。

李嘉图发展为**比较优势**——即使一国在所有商品生产上都处于绝对劣势，仍可通过专业化生产其相对劣势最小的商品而获益。

#### 贸易利益的来源

```
英国以100人的劳动产品换取葡萄牙80人的劳动产品

这不会发生在同一国家的个人之间：
- 100个英国人的劳动不能与80个英国人的劳动相交换

但可以发生在国际间：
- 100个英国人劳动的产品能与80个葡萄牙人的劳动产品交换
```

### 二、货币与汇率

#### 货币价值决定

> **金银的价值大小只根据其生产和将其运送到市场上的必要劳动量的多少**

#### 汇率与货币分配

货币通过国际贸易在各国间分配，其分配比例应能适用于假定不存在金银、国际贸易纯粹是易货贸易时所出现的自然交易情况。

#### 货币流动机制

```
贸易顺差国 → 货币流入 → 货币价值下降 → 商品价格上升 → 出口减少、进口增加 → 货币流出
贸易逆差国 → 货币流出 → 货币价值上升 → 商品价格下降 → 出口增加、进口减少 → 货币流入
```

最终，各国的货币量会调整到使汇率恢复平价。

## 第四篇 赋税理论

### 一、赋税的基本原则

#### 税收来源

> **赋税是一个国家土地和劳动的产品中由政府支配的部分，它最终是从该国的资本或收入中支出的。**

#### 税收对积累的影响

```
税收 → 要么落在资本上 → 减少生产规模
      → 要么落在收入上 → 减少积累或非生产性消费
```

**李嘉图的重要论断**：一切税收都能减弱积累能力。

### 二、各种税收的影响分析

#### 农产品税

| 效应 | 说明 |
|------|------|
| 提高农产品价格 | 数额与税额相等 |
| 提高劳动工资 | 降低利润 |
| 由消费者负担 | 地主和农场主不直接负担 |

**数学示例**：
```
假设每夸脱征收8先令税：
- 谷物价格从4英镑涨至4英镑8先令
- 货币地租不变（差额不变）
- 谷物地租下降（差额比例变化）
```

#### 地租税

- **只影响地租**
- **完全由地主负担**
- **不能转嫁给任何消费者**

原因：地主无法改变最差土地与优等土地之间的差额。

#### 什一税

- **对土地总产量征收**
- **完全由消费者负担**
- **不同于地租税，影响所有等级土地**
- **是土地总产品的1/10，而非净产品的1/10**

**特点**：随社会进步，净产品在总产品中的比例减少，什一税在净产品中所占比例越来越大。

#### 工资税

> **工资税使工资上涨，因而减少资本利润率**

工资税与必需品税的区别：
- 必需品税必须伴随必需品价格上涨
- 工资税不一定（可能劳动者满足于较少享乐品）

**最终结论**：影响工资的任何税都是通过**减少利润**的方式缴纳的，工资税实际上是利润税。

#### 利润税

对所有行业利润均按比例征税：
- 所有商品价格按同等程度涨价
- 相对价值保持不变
- 每人都以相同比例从收入中拿出一部分

### 三、赋税对各阶级的影响

#### 地主阶级

- 农产品税：通过农产品涨价转嫁给消费者
- 地租税：**直接负担，不能转嫁**
- 什一税：通过农产品涨价转嫁

#### 资本家阶级

- 工资税：**通过利润减少承担**
- 农产品税：通过支付更高工资承担
- 利润税：直接承担

#### 劳动者阶级

- 不能承受繁重赋税
- 最低工资不能长期超过维持生理和习惯所需
- 任何对必需品的征税最终都影响他们

### 四、出口补贴与进口禁令

#### 出口补贴的效果

> **谷物出口补贴的最后结果不是提高或降低国内市场的谷物价格，而是降低了国外消费者的谷物价格。**

补贴的影响链条：
```
补贴 → 出口有利 → 需求扩大 → 短期国内价格上涨
     → 农业利润提高 → 资本流入农业
     → 供给增加 → 价格恢复到自然水平
```

#### 进口禁令的影响

李嘉图批评英国的《谷物法》：

> **禁止进口谷物使英国消费者支付的谷物价格远高于自由贸易时的价格，这仅有利于地主阶级，而损害了社会其他阶层。**

## 第五篇 价值与财富的区分

### 一、核心区分

李嘉图提出了价值与财富的本质区别：

| 概念 | 定义 | 决定因素 |
|------|------|---------|
| **价值** | 交换能力 | 生产的困难或便利程度（劳动量） |
| **财富** | 必需品、享用品、娱乐品 | 商品的充足程度 |

### 二、机器对财富的影响

#### 重要发现

> **机器的发明和使用可能伴随着总产值的减少。每当出现这种情况时，劳动者就会受到伤害，因为他们当中部分人会失业。**

这一发现与李嘉图早期观点形成对比，表明他对劳动者利益关注的深化。

#### 分析框架

```
改进前：
- 资本20000英镑（7000固定+13000流动）
- 产值15000英镑
- 养活工人13000英镑

改进后（使用机器）：
- 固定资本增加
- 流动资本减少
- 净产值可能增加
- 但总产值可能减少
- 劳动需求减少
```

### 三、纯收人与总收入

#### 区分的重要性

> **一国维持海陆军和非生产性劳动的能力必须与其纯收入而非总收入成正比。**

```
总收入：2000万
必要生产开支：1000万（劳动者必需品）
纯收入：1000万（可纳税和储蓄）
```

## 核心概念工具箱

### 1. 劳动价值论分析框架

```
商品价值 = 生产所需劳动量（直接+间接）

影响因素：
- 直接劳动量变化 → 价值同比例变化
- 固定资本比例变化 → 相对价值变化
- 工资变动 → 影响使用固定资本多的商品价格
```

### 2. 级差地租计算模型

```
产量差额法：
- 确定边际土地（不付地租）
- 各等级土地地租 = 该土地产量 - 边际土地产量

或：
- 确定价格 = 边际土地成本（劳动+利润）
- 地租 = 实际产量价值 - 边际土地等量产量价值
```

### 3. 工资-利润-地租三角关系

```
总产品分配：
┌─────────────┐
│    地租     │ ← 随人口增加而增加
├─────────────┤
│    利润     │ ← 随工资增加而减少
├─────────────┤
│    工资     │ ← 随必需品价格波动
└─────────────┘
```

### 4. 比较优势决策矩阵

```
各国选择专业化方向：
- 比较各国生产同种商品的相对成本
- 专业生产相对成本最低的商品
- 通过贸易获得其他商品

判断标准：放弃某种商品生产时，损失最小
```

### 5. 赋税归宿分析框架

```
税收最终由谁负担？
┌─────────────────────────────┐
│ 1. 需求弹性小的商品 → 消费者负担 │
│ 2. 竞争激烈的市场 → 生产者负担   │
│ 3. 不可替代的生产要素 → 该要素负担│
└─────────────────────────────┘

工资税 → 实际由利润承担
地租税 → 实际由地主承担
农产品税 → 消费者承担
```

## 理论联系实际

### 一、对政策制定的启示

1. **谷物政策**
   - 自由贸易优于保护主义
   - 进口禁令损害消费者利益
   - 地主是保护政策的主要受益者

2. **税收政策**
   - 税收最终会影响资本积累
   - 应避免对资本积累的过度干扰
   - 平等课征优于选择性征税

3. **货币政策**
   - 纸币发行需有适当限制机制
   - 货币价值受生产条件影响
   - 货币数量与商品价格相互影响

### 二、对理解当代经济的价值

李嘉图的理论框架对于理解以下当代问题仍有启发：

| 当代议题 | 李嘉图理论的应用 |
|---------|-----------------|
| 全球化与贸易 | 比较优势原理仍是对外贸易理论基础 |
| 收入分配 | 工资-利润对立关系的现代诠释 |
| 技术进步与就业 | 机器替代劳动对就业的影响 |
| 税收归宿 | 间接税与直接税对各阶级的影响分析 |
| 地租与房价 | 级差地租理论的城市应用 |

## 延伸阅读

### 1. 《国富论》—— 亚当·斯密

> **推荐理由**：李嘉图的理论是对斯密理论的发展与修正。阅读斯密的原著可以理解古典政治经济学的起点，以及李嘉图批判继承的具体内容。

### 2. 《资本论》—— 卡尔·马克思

> **推荐理由**：马克思批判继承了李嘉图的劳动价值论，建立了剩余价值理论。了解李嘉图的理论有助于理解马克思主义政治经济学的理论来源。

### 3. 《政治经济学原理》—— 马尔萨斯

> **推荐理由**：马尔萨斯是李嘉图同时代的重要经济学家，两人在学术上多有争论。阅读马尔萨斯的著作可以理解当时经济学界对李嘉图理论的回应。

### 4. 《人口原理》—— 马尔萨斯

> **推荐理由**：李嘉图在本书中多次提及马尔萨斯的人口理论。理解人口原理对于把握李嘉图分配理论的前提假设至关重要。

### 5. 《经济学原理》—— 马歇尔

> **推荐理由**：马歇尔综合了古典经济学和新古典经济学的成果，代表了经济学发展的下一个阶段。从马歇尔的著作可以了解李嘉图理论的后续发展。

### 6. 《就业、利息和货币通论》—— 凯恩斯

> **推荐理由**：凯恩斯革命批判了古典经济学的若干假设，包括李嘉图关于工资和就业的部分观点。理解凯恩斯的批评有助于深化对李嘉图理论局限性的认识。

---

## 总结

《政治经济学及赋税原理》是经济学说史上最重要的著作之一。李嘉图在本书中：

1. **系统发展了劳动价值论**，确立了劳动作为价值尺度的地位
2. **建立了级差地租理论**，揭示了地租产生的真正原因
3. **阐明了工资与利润的对立关系**，论证了利润下降的趋势
4. **发展了比较优势原理**，为自由贸易理论奠定基础
5. **分析了各种税收的归宿**，揭示了税收对经济的影响

尽管李嘉图的理论存在若干局限（如劳动价值论的适用边界、静态分析的局限等），但其分析框架和方法论贡献至今仍深刻影响着经济学的发展。马克思称其为"资产阶级古典政治经济学的完成者"，实至名归。
