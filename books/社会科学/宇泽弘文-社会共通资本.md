# 社会共通资本

## 核心思想概述

宇泽弘文是20世纪最具原创性的经济学家之一，在一般均衡理论、经济增长理论、动态经济学等领域做出奠基性贡献后，将目光转向一个更具根本性的问题：**人类应当建立怎样的经济体制，才能超越资本主义与社会主义的对立，实现真正富足的社会？**

本书的核心论点是：**社会共通资本（Social Overhead Capital）** 是理解理想经济体制的关键概念，它源自托斯丹·凡勃伦的制度主义思想，经由约翰·杜威的自由主义哲学演化而来。宇泽认为，20世纪末无论是资本主义阵营还是社会主义阵营，都陷入了深刻的危机，其根本原因在于**未能有效管理和维护各自的社会共通资本**。

## 第一部分：理论基础

### 一、制度主义的思想谱系

#### 1.1 凡勃伦的制度主义

凡勃伦在19世纪末提出制度主义，核心主张是：

- 经济体制不是从抽象逻辑演绎出来的，而是由每个国家或地区的**伦理、社会、文化及自然环境**等多重条件相互交织而成
- 经济制度会随经济发展阶段和社会意识变革而变化
- 制度主义否定了马克思主义将生产关系作为决定性因素的观点，也超越了新古典经济学脱离社会文化条件追求理想制度的研究路径

凡勃伦提出的两个关键概念：
- **"自由求知欲"（idle curiosity）**：人类本能地追求知识本身，而非知识带来的物质效用
- **"匠人气质"（instinct of workmanship）**：技术人员和劳动者本能地追求最佳生产方式，但在追求利润的外部动机支配下往往无法实现

#### 1.2 从社会主义到资本主义的世纪困境

1991年，罗马教皇约翰·保罗二世发布新版《新事》通谕，重申"资本主义的弊害和社会主义的幻想"这一百年前的命题。三个月后，苏联"八一九事件"爆发，社会主义阵营崩溃。

**关键洞察**：马克思设想的从资本主义到社会主义的历史发展进程，在现实中发生了逆转——东欧各国从社会主义转向资本主义。但是：

| 计划经济制度的内在矛盾 | 市场经济制度的内在矛盾 |
|:---|:---|
| 国家权力不受约束 | 收入分配不公平，代际积累加剧不平等 |
| 公民权利受限 | 投机性动机支配生产性动机 |
| 与个人内在动机根本矛盾 | 市场均衡稳定性假设不符合现实 |

凯恩斯已证明，纯粹市场机制无法实现充分就业；凡勃伦则揭示了企业作为"实体性组织"的特征，与新古典经济学中企业可自由调整生产要素组合的假设相矛盾。

#### 1.3 经济理论的两次危机

琼·罗宾逊（Joan Robinson）于1970年提出"经济理论的第二次危机"：

| 第一次危机（1930年代） | 第二次危机（1970年代） |
|:---|:---|
| 大萧条暴露新古典经济学的理论缺陷 | 越战引发资本主义结构性矛盾 |
| 凯恩斯主义应运而生 | 凯恩斯主义失效，无新范式替代 |

然而，1970-80年代的经济学发展却背道而驰——货币主义、供给经济学、理性预期假说等反凯恩斯主义思潮兴起，以更极端的形式复活了新古典经济学。宇泽称之为"历史的逆流"。

### 二、社会共通资本的概念框架

#### 2.1 什么是社会共通资本

**定义**：社会共通资本是一种社会结构，能够使一个国家或地区的人过上富足的生活，让他们的优秀文化得到发扬和传承，还能够持续、稳定地维持一个充满人性魅力的社会。

**三个基本特征**：
1. 让人的尊严和灵魂的独立性得到保护和尊重
2. 最大限度地保障公民的基本权利
3. 即使由私人所有或管理，也必须作为全社会共有的财产，遵循社会性标准管理和运用

#### 2.2 三种类型

```
┌─────────────────────────────────────────────────────────┐
│                   社会共通资本                            │
├─────────────────┬─────────────────┬─────────────────┤
│    自然环境       │   社会基础设施    │    制度资本      │
├─────────────────┼─────────────────┼─────────────────┤
│ 空气            │ 道路交通设施      │ 教育制度         │
│ 森林            │ 上下水管道        │ 医疗制度         │
│ 河流、湖泊      │ 电力、燃气        │ 司法制度         │
│ 海洋、湿地      │ 公共交通          │ 金融制度         │
│ 土壤            │ 通信设施          │ 行政制度         │
└─────────────────┴─────────────────┴─────────────────┘
```

#### 2.3 管理的根本原则

**核心原则**：社会共通资本必须基于各领域**专家的专业知识**，遵照**职业规范**进行管理和运营，绝不能：

- 按照政府规定的官僚标准管理
- 遵循市场标准（追求利润最大化）

这要求基于**信托（fiduciary）原则**进行管理：受托机构必须保持独立自主，基于专业知识和职业规范行事，对全体公民直接负有责任。

#### 2.4 富足社会的条件

1. 稳定、持续地维持美好、富足的自然环境
2. 提供整洁的居住环境和舒适生活的文化环境
3. 建立良好的学校教育制度，让所有孩子发挥多样化禀赋
4. 身患疾病者能随时接受最高水平的医疗服务
5. 拥有完善的经济和社会制度，实现高效、均衡的资源分配

## 第二部分：领域分析

### 三、农业与农村

#### 3.1 农耕的本质特征

宇泽主张用"农耕"而非"农业"来思考问题：

| 特征 | 含义 |
|:---|:---|
| 历史与人类同样久远 | 是人类的根本特征 |
| 与自然和谐共存 | 遵循自然规律，不会带来大规模自然破坏 |
| 避免"自我异化" | 劳动者可维持人格同一性，在自然怀抱中自由生存 |
| 环境保护功能 | 水田耕作能抑制甲烷排放，调节气温 |

#### 3.2 日本《农业基本法》的错误

1961年制定的《农业基本法》的基本思想存在严重问题：

- **错误假设**：农业与工业具有相同性质，可用市场效率标准衡量
- **政策导向**：追求农业与工业部门竞争的条件
- **实际结果**：
  - 农户数量从600万降至320万（38年间减少近一半）
  - 主要从事农业的人口从1200万降至230万
  - 65岁以上农业从业者占比从7%升至45%
  - 应届毕业生从事农业者从9万人降至1800人

#### 3.3 农业与工业的非对称性

新古典经济学主张土地稀缺的国家应缩小农业、侧重工业。但这一理论存在根本谬误：

1. **"经济福利"概念模糊**：假设代表性个人用边沁式效用评价，忽视了收入分配不平等
2. **生产要素"可变性"假设脱离现实**：农业生产要素无法像理论假设那样无成本转换
3. **无视农业的外部经济效应**：环境保护、生物多样性维持、文化传承等功能无法纳入市场价格

#### 3.4 "公地"思想与三里塚农社构想

**公地（commons）的真实含义**：
- 并非"开放进入"的无主资源
- 由特定集团或社区"共有"，有传统规范约束使用方式
- 哈丁的"公地悲剧"论述基于对新古典经济学假设的误用

**三里塚农社的制度设计**：

农社由社员组成，设立四个部门：
- **大学部**：全盘规划和经营
- **农场部**：可持续农业发展实践
- **工厂部**：农产品加工和销售
- **建设部**：建筑设施建设和维护

关键特征：
- 公有土地与私有土地并存
- 民主主义决策机制
- 农业生产与加工、销售、研究一体化
- 对自然和社会环境无害的生产形态

### 四、城市

#### 4.1 20世纪城市规划的失败

勒·柯布西耶的"光辉城市"理念：
- 将城市视为艺术作品，追求几何和抽象之美
- 以合理主义精神为基础
- 人在其中只是不具有主体性的"机器人"

**问题**：20世纪城市规划完全忽视了居住和生活的人，只是城市规划者自身片面、单调的人性特征的投影。

#### 4.2 简·雅各布斯的批判与建设性贡献

《美国大城市的死与生》（1961）提出四项原则：

| 原则 | 核心内容 |
|:---|:---|
| 街道空间 | 尽量缩小街道宽度，道路以到达每个街区距离最短为佳，可蜿蜒曲折 |
| 建筑保留 | 再开发时应尽可能保留古旧建筑，避免高折旧费阻碍自由灵感 |
| 功能多样性 | 各区域必须具备两个或以上功能，否定分区规划思想 |
| 人口密度 | 尽可能提高各区域人口密度 |

#### 4.3 汽车的社会性费用

宇泽早在1974年出版的《汽车的社会性费用》中系统分析了这一问题。

**汽车社会性费用的四个构成要素**：

1. **道路使用的机会成本**：行人、儿童失去安全使用街道的权利

2. **交通事故的生命健康损失**：
   - 日本道路结构不适合汽车通行
   - 非专业人员驾驶增加事故概率
   - 霍夫曼制度（以未来收入现值评估生命）缺乏人性

3. **污染和环境破坏**：
   - 大气污染（NO₂与悬浮颗粒物引发肺癌）
   - 噪音、震动威胁健康
   - 西淀川公害诉讼（1978-1998）的历史意义：确立了国家和道路集团对大气污染的共同责任

4. **对自然和文化环境的破坏**：
   - 森林枯萎、地形平衡破坏
   - 城市空间被汽车过度占据
   - 社会环境不稳定、文化环境恶劣

**核心论点**：社会性费用未被内部化，导致对汽车需求无限膨胀，造成恶性循环。

### 五、学校教育

#### 5.1 教育的本质

**定义**：帮助每个孩子尽可能发挥先天及后天资质，幸福地度过一生。

**关键原则**：
- 不能以单一尺度衡量孩子的能力
- 不能以排名形式评价孩子表现
- 必须尊重每个孩子的独特禀赋

#### 5.2 乔姆斯基的发现与教育

每个孩子在语言和数学上具有**先天的（innate）知识和能力**。以平面几何为例：

- 孩子能直观理解"两点间距离在运动中保持不变"
- 这蕴含着二维欧几里得空间的复杂性质
- 教学必须尊重这种先天理解力，而非强制灌输形式化定义

**核心比喻**：孩子的先天理解力如"花蕾"，教育的目的就是施以适当刺激，助其成长开花，但花蕾极为纤细，绝不能粗暴对待。

#### 5.3 杜威的教育理论与鲍尔斯-金迪斯的批判

**杜威的三个教育功能**：
1. **社会统合**：把孩子培养成能对社会做贡献的成员
2. **平等主义**：纠正社会、经济体制的不平等
3. **促进人格成长**：发展独特的身体、智力、情绪、审美潜能

**鲍尔斯-金迪斯的"对应原理"批判**：
- 在法人资本主义体制下，学校教育无法统一发挥三个功能
- 学校制度复制了经济体制的不平等，而非解决它
- "IQ遗传论"（詹森、赫恩斯坦）在统计学上站不住脚
- **结论**："压抑和不平等的根源存在于资本主义的经济结构和功能之中"

#### 5.4 凡勃伦的大学论

凡勃伦在《美国的高等教育》（1918）中批评商业思维主导大学：

**大学的本质**：
- 人类基于"自由求知欲"和"匠人气质"追求积累深奥知识的场所
- 研究学问是首要功能，教育学生是第二位
- 知识本身具有固有价值，不应以市场标准评价

**危机**：商业标准被引入大学——用"生产"多少有用知识、培养学生数量来评价研究人员，按利润最大化原则经营。

#### 5.5 英国大学制度的启示

**大学基金委员会（UGC）的设计**：
- 从财政部获得预算，再分配给各大学
- 各大学可自由支配预算
- 代表大学向政府主张，不传达政府意志

**剑桥大学的"学院制"**：
- 各学院拥有丰厚基金，人事财务完全独立
- 资深顾问制度：登门拜访每位申请者，判断其人格成长可能性
- 保护学术自由和学生人格发展的制度设计

### 六、医疗

#### 6.1 医疗作为社会共通资本的原则

**政府的义务**：
- 构建医院体系
- 培养医疗专业人员
- 提供原则上免费或低廉的保健医疗服务

**分配标准**：不能用官僚方式或市场方式确定，必须基于医学专业知识，以同行评议等方式检验医疗专家的职业能力和人格资质。

#### 6.2 现行诊疗报酬制度的问题

**保险计分制的弊端**：
- 以物质成本为核心，低估医生、护士等技术成本
- 完全忽略医生经验和技术水准
- 技术水准低的医生造成手术失败反而获得更多计分
- 护士技术工资被压低，劳动强度极高

**"医疗最优与效益最优的背离"**：
- 从医学角度看最优的医疗行为，效益上可能达不到期望
- 检查费、药费的盈余填补人工费用的赤字，形成扭曲激励

#### 6.3 静态与动态分析

| 分析维度 | 核心问题 |
|:---|:---|
| **静态分析** | 给定医疗设施和人员，应实施哪些诊疗？如何负担费用？医生的判断不能受费用影响 |
| **动态分析** | 如何建设医疗设施？如何积累医疗资本？如何培养人才？ |

**关键结论**：
- 不能依据市场标准决定医生收入（否则会偏向高收益患者）
- 大部分报酬应是固定工资，绩效工资应尽量限制
- 从社会、文化角度看，用于医疗的人力物力资源越多越好

### 七、金融制度

#### 7.1 20世纪末金融危机与30年代大萧条的比较

| 1920年代-1930年代 | 1980年代-1990年代 |
|:---|:---|
| 土地、股票投机泡沫 | 金融自由化、取消监管 |
| 银行以10%保证金煽动投机 | S&L被允许参与广泛金融业务 |
| 《格拉斯-斯蒂格尔法案》分离银行与证券 | 放松监管导致金融机构冒险 |
| 失业率达37% | 日本金融危机使经济停滞数十年 |

**罗斯福新政的启示**：
- 《1933年银行法》将银行作为社会共通资本，设定社会标准
- 田纳西河流域管理局（TVA）作为社会基础设施建设的典范

#### 7.2 货币主义和理性预期假说的批判

**理性预期假说的根本矛盾**：
- 假设人们能准确获知需求曲线和供给曲线
- 这要求深入了解所有经济主体的偏好、技术、动机
- 这**否定了分权制**——而分权制正是市场制度存在的前提

**结论**：理性预期假说在逻辑上自我矛盾，却为"市场机制始终高效"的新古典信条提供了"理论依据"。

#### 7.3 日本金融崩溃的教训

**住专问题的根源**：
- "护航制度"导致金融机构缺乏职业操守
- 泡沫经济期间大型银行主导大规模土地收购
- 破坏了农村、山区的生态和人文环境

**深层问题**：
- 金融机构将公有资本用于投机而非服务实体经济
- 金融制度走上国际化道路后，管理难度倍增
- 社会共通资本不能被市场左右，也不能被官僚管理

### 八、地球环境

#### 8.1 自然环境的经济学定位

**自然环境的特性**：
- 类似于经济理论中的**存量**概念
- 再生产过程取决于生物学/生态学因素
- 时间变化率在本质上不同于工业资本

**森林的例子**：
- 面积扩大两倍，木材产量不会成比例增加
- 多种要素（水流、土壤、生物共生关系）相互影响
- 气象条件使经济功能具有统计学和概率论意义

#### 8.2 传统社会与近代社会的文化差异

| 传统社会 | 近代社会 |
|:---|:---|
| 文化包括行为模式、技术、信念、制度 | 文化仅限定于知识性、艺术性活动 |
| 自然与文化、宗教浑然一体 | 自然被当作可征服、支配的对象 |
| 可持续利用自然资源 | 无限索取导致环境破坏 |

**西雅图酋长的警示**：
> "白人对大地予取予求，每一寸大地对他们而言看来都是一样的。他将大地视为敌人，一步一步地加以征服。"

#### 8.3 两次国际环境会议

| 1972年斯德哥尔摩会议 | 1992年里约热内卢会议 |
|:---|:---|
| 主题：环境公害问题 | 主题：全球环境污染与破坏 |
| 二氧化硫、有机水银等工业废物 | 全球气候变暖、生物多样性威胁 |
| 瑞典酸雨问题、日本水俣病 | 热带雨林骤减 |
| 工业活动直接损害健康 | 温室气体影响未来世代 |

#### 8.4 全球气候变暖的经济学分析

**温室效应的机制**：
- 太阳表面6000°C，地球表面15°C
- 温室气体（CO₂、甲烷、氟利昂等）吸收红外辐射
- 无温室气体：地球均温-18°C
- 温室气体过多（如金星）：均温470°C

**工业革命以来**：
- CO₂浓度从280ppm升至350ppm（增加25%）
- 热带雨林减少导致光合作用吸收效率降低

#### 8.5 碳税制度设计

**宇泽的比例碳税思想**：

传统经济学家提议的"一刀切"碳税（每吨碳100美元）：
| 国家 | 人均国民收入 | 应缴碳税 | 占比 |
|:---|:---|:---|:---|
| 日本 | 19,000美元 | 250美元 | 1.3% |
| 美国 | 17,000美元 | 340美元 | 2.0% |
| 印度尼西亚 | 400美元 | 30美元 | **7.5%** |
| 菲律宾 | 500美元 | 60美元 | **12%** |

**问题**：对发展中国家造成不成比例的负担，可能阻碍其经济发展。

**比例碳税方案**：按人均国民收入的一定比例设定税率，实现国际公平与代际公平的统一。

#### 8.6 大气稳定化国际基金

**运作机制**：
1. 各国政府征收比例碳税
2. 扣除植树造林补助金后，抽取5%存入基金
3. 按人口和人均国民收入分配给发展中国家
4. 资金用于热带雨林保护、农村维护、新能源开发

**瑞典的示范**：1991年成为首个引入碳税国家，税率高达每吨碳150美元，碳税是更大范围税制改革的一部分，经过民主程序审议通过。

#### 9.2 超越资本主义与社会主义

宇泽的制度主义经济体制的特征：
- 不从普遍、统一原理演绎
- 由各国家/地区的具体条件相互交织而成
- 会随经济发展阶段和社会意识变革而变化
- 以**社会共通资本**的存在方式为核心

**最终指向**：实现一种近乎理想的经济体制，使人们能够最大限度享受作为公民的基本权利，同时保持人的尊严和灵魂的独立性。

### 十、日本"世纪末断层"的诊断与处方

#### 10.1 问题诊断

日本面临的世纪末式混乱的共同原因：
- 学校教育：引入市场原理，导致应试地狱和精神荒芜
- 农业：错误的《农业基本法》导向农业衰退
- 城市：汽车社会的无节制扩张
- 环境：经济增长代价是自然破坏
- 金融：监管放松导致泡沫与崩溃

**根本病因**：未认清教育、农业等都是社会共通资本，却用市场原理或官僚管理对待它们。

#### 10.2 改革方向

| 领域 | 核心改革 |
|:---|:---|
| 学校教育 | 建立保障人格成长的制度，守护大学自由 |
| 农业 | 建立"公地"性质的农社，保障农村合理规模 |
| 城市 | 采纳雅各布斯式城市规划，优先步行和公共交通 |
| 医疗 | 建立医学最优与效益最优相一致的制度 |
| 金融 | 明确职业规范，将金融作为社会共通资本管理 |
| 环境 | 实施比例碳税，建立大气稳定化国际基金 |
