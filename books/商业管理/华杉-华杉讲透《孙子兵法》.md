# 华杉讲透《孙子兵法》

## 核心导读

《孙子兵法》是中华文明智慧皇冠上最璀璨的宝石，作者华杉以企业家和管理者的视角，对这部两千五百年前的兵学圣典进行了深入浅出的注解。全书以《十一家注孙子》为基础，继承曹操、杜牧、陈皞等十一位古代注家的研究成果，并参照郭化若（中国人民解放军中将）的现代译注，形成了自己独特而完整的解读体系。

**本书的核心宗旨**：不是教你如何打仗，而是教你如何不战而胜；不是教你如何算计别人，而是教你如何管好自己。

## 第二部分：战略总纲——“五事七计”

### 一、五事：战争评估的五个维度

**道——软实力**

“令民与上同意也”，让人民与君主上下一心。这是战争的政治基础。

- 不是战争的正义性，而是人心向背
- 刘邦“约法三章”对项羽的对比
- **软实力决定根本胜负**

**天——军事气象学**

寒暑四时、阴阳变化。兵家讲天，主要是气象条件，不是占卜星象。

**地——地形地势**

远近、险易、广狭、死生。地形是作战的关键辅助条件。

**将——将领素质**

智、信、仁、勇、严五德具备，尤以“智”为首。

**法——组织管理**

曲制（编制）、官道（人事）、主用（后勤）。组织架构与管理是“军之大事”。

### 二、七计：具体的胜负评估

1. **主孰有道** —— 哪方君主更得人心
2. **将孰有能** —— 哪方将领更有能力
3. **天地孰得** —— 哪方占据天时地利
4. **法令孰行** —— 哪方法令执行更严明
5. **兵众孰强** —— 哪方军队战斗力更强
6. **士卒孰练** —— 哪方士兵训练更充分
7. **赏罚孰明** —— 哪方赏罚更分明

**计算的结果就是胜负**。先胜后战，不胜不战。

## 第三部分：作战成本——“战争经济学”

### 一、打仗就是打钱粮

《作战篇》详细计算了战争成本：

- 驰车千驷、革车千乘、带甲十万
- 日费千金
- 千里馈粮，运粮成本高达20倍

**“兵贵胜，不贵久”**——战争拖得越久，消耗越大，国家财政难以支撑。

### 二、历史教训：汉武大帝的代价

汉武帝开疆拓土、“明犯强汉者，虽远必诛”的辉煌背后：
- 国库掏空
- 盐铁专营、任意加税
- 向中产以上人家收取财产税
- 全国举报制度导致人人自危
- **晚年发布《轮台罪己诏》自我检讨**

### 三、核心原则：利害计算

- 先考虑避害，再考虑趋利
- 利害权重应为50:50，但人们往往趋利忘害
- **李嘉诚法则**：做任何事情先考虑失败

## 第四部分：谋攻之道——“全胜”思想

### 一、全胜的战略层次

| 层次 | 内容 |
|------|------|
| 全国为上 | 完整接收敌国 |
| 破国次之 | 军事占领敌国 |
| 全军为上 | 完整收编敌军 |
| 破军次之 | 军事击败敌军 |

**不战而屈人之兵，善之善者也** ——这是孙子追求的最高境界。

### 二、伐谋与伐交

- **上兵伐谋** —— 破坏敌方计谋，使其放弃企图
- **其次伐交** —— 瓦解敌方外交联盟

**典型战例**：晏子“折冲樽俎”，在酒席谈判中化解齐晋冲突。

### 三、兵力原则

- 十则围之
- 五则攻之
- 倍则分之
- 敌则能战之
- 少则能逃之
- 不若则能避之

**“小敌之坚，大敌之擒”** —— 以卵击石必败，该跑就跑。

---

## 第五部分：军形与兵势——“形势”二篇

### 一、军形篇：先为不可胜

**核心思想**：先让自己不可战胜，等待敌人出现可胜之机。

> “不可胜在己，可胜在敌”——管好自己，等待敌人犯错。

**修道保法**——修炼自身实力和法度。

### 二、兵势篇：以正合，以奇胜

**“奇”念jī，不念qí，是余奇、预备队的意思。**

- 正兵先交战
- 奇兵在关键时刻打出
- 正奇可以相互转换

**韩信背水一战**的真正秘密：不是“置之死地而后生”，而是分战法——有正有奇，两千奇兵夺营才是胜负手。

### 三、造势与任势

- **气势** —— 内心强大，众人信服
- **地势** —— 占据有利地形
- **因势** —— 根据敌人情况相机行事

**“求之于势，不责于人”** —— 领导者要造势，不要苛责下属。

---

## 第六部分：虚实之道——“避实击虚”

### 一、核心原理

**“兵之形，避实而击虚”**

- 我专而敌分
- 以十击一
- 敌所备者多，则吾所与战者寡

### 二、虚实的关键

**“致人而不致于人”** —— 调动敌人，而不被敌人调动。

**五种调动敌人的方法**：
1. 利而诱之
2. 乱而取之
3. 实而备之
4. 强而避之
5. 怒而挠之

### 三、毛泽东的运用

**“集中优势兵力打歼灭战”**——完全符合孙子虚实思想。

## 第八部分：用间之道——“情报战”

### 一、五种间谍

| 间谍类型 | 定义 | 特点 |
|---------|------|------|
| 因间 | 敌方乡人 | 利用当地人 |
| 内间 | 敌方官员 | 利用官吏 |
| 反间 | 敌方间谍 | 反过来用 |
| 死间 | 传递假情报 | 可能牺牲 |
| 生间 | 活着回报 | 正常情报 |

### 二、反间是核心

**“五间之事，知之必在于反间”**

- 必须找到敌方间谍并策反
- 重金收买，恩义相结
- “非圣智不能用间，非仁义不能使间”

### 三、伊尹与姜子牙的启示

商周的兴起，都依靠“从敌人内部出来的人”。**“以上智为间者，必成大功”**。

## 第九部分：领导力——“将道”

### 一、将帅五德

**智、信、仁、勇、严**

| 德目 | 含义 |
|------|------|
| 智 | 智能识权变 |
| 信 | 赏罚分明 |
| 仁 | 爱兵如子 |
| 勇 | 当机立断 |
| 严 | 以威肃众 |

### 二、曾国藩的补充：“廉”和“明”

- **廉** —— 账目公开，清廉服众
- **明** —— 把每个人的功劳过错看得分明

**“将将之道”与“将兵之道”** —— 领导者要把每个人的作用都看清楚。

### 三、领导者的五个性格缺陷

| 缺陷 | 后果 |
|------|------|
| 必死 | 可杀 |
| 必生 | 可虏 |
| 忿速 | 可侮 |
| 廉洁 | 可辱 |
| 爱民 | 可烦 |

## 第十部分：治军之道——“分数与形名”

### 一、组织架构

**“分数”** —— 编制问题
- 班、排、连、营、团、师
- 林彪“三三制”：最小单位三人，逐级指挥

**“形名”** —— 指挥系统
- 旌旗是视觉号令
- 金鼓是听觉号令

### 二、治气、治心、治力、治变

| 治法 | 内容 |
|------|------|
| 治气 | 避其锐气，击其惰归 |
| 治心 | 以治待乱，以静待哗 |
| 治力 | 以近待远，以佚待劳 |
| 治变 | 无邀正正之旗，勿击堂堂之阵 |

### 三、厮养之道

**“卒未亲附而罚之，则不服；卒已亲附而罚不行，则不可用”**

恩威并施，刚柔相济。

## 第十一部分：现代管理启示

### 一、SWOT分析与管理决策

《孙子兵法》的“五事七计”相当于现代的SWOT分析：
- Strengths（优势）→ 己方强项
- Weaknesses（劣势）→ 己方弱项
- Opportunities（机会）→ 可乘之机
- Threats（威胁）→ 敌之实

### 二、竞争战略

**“先胜后战”** = 先比较实力，再决定行动

| 情况 | 策略 |
|------|------|
| 实力占优 | 正面进攻 |
| 实力相当 | 分化对方 |
| 实力不足 | 等待/转移 |

### 三、组织管理

**“治众如治寡，分数是也”** —— 组织架构决定管理效率

- 层级设计
- 权责分明
- 赏罚清晰

## 第十二部分：核心心法总结

### 一、孙子的敬畏心

> “兵者，国之大事，死生之地，存亡之道，不可不察也。”

这不仅是对军事家说的，对每个人的人生都适用。

### 二、四个关键心态

1. **不能等待** —— 是巨大的性格缺陷
2. **贪巧求速** —— 是人性弱点
3. **侥幸心理** —— 自我欺骗
4. **焦虑动作** —— 最能毁自己

### 三、成功者的共同特质

- 懂得等待
- 甘于平凡
- 持续积累
- 二十年磨一剑

> **“善战者无智名，无勇功”** —— 真正的成功者，都是在没人注意的地方默默耕耘。

**《孙子兵法》的最终启示**：

> **先胜后战，赢了再打。不胜不战，待时而动。**

这是兵法的精髓，也是人生的智慧——不是教你如何打赢，而是教你如何不战。
