# 依恋

> **作者**：約翰·鮑比（John Bowlby）
> **核心命題**：人類嬰兒與照顧者（尤其是母親）之間的依附關係，是人格發展與心理健康的核心基礎，其根源深植於演化本能，而非僅由生理需求（如飢餓）所驅動。

## 二、核心概念：本能行為的組織原理

### 2.1 進化適應性環境

鮑比引入「進化適應性環境」（Environment of Evolutionary Adaptedness, EEA）概念，作為理解人類本能行為的關鍵框架：

> 「人類本能構造的適應性環境，應當探討的是在幾千年前一直到兩百萬年前人類居住地驚人的多樣化發展。」

這意味著：

- 人類文明環境（尤其是近200年的工業化）與進化適應性環境極度不一致
- 人類基本社會單位是母親及其孩子，而非更大的群體
- 理解人類行為必須回溯到祖先環境中這些行為曾經服務的功能

### 2.2 行為系統理論

鮑比用「行為系統」概念替代傳統的「驅力」與「能量釋放」：

**核心特徵：**

| 概念 | 定義 |
|------|------|
| **設定目標**（Set-goal） | 行為系統所達成的可預期結果 |
| **負回饋** | 環境變化產生的信號終止當前行動 |
| **終止條件** | 特定信號觸發行為停止，而非能量耗竭 |

例如，攝食行為的終止條件是「胃部充滿食物」（紅燈終止車流），而非「能量耗竭」（汽油用盡）。

**環境穩定型 vs. 環境依賴型系統：**

- **環境穩定型**：受環境多樣性影響較少，如鳥的築巢行為
- **環境依賴型**：成年期形式極大取決於成長環境
- 人類的依附系統高度環境依賴，這解釋了為何文化差異會顯著影響依附行為的具體表現

### 2.3 本能行為的組織層級

鮑比區分了本能行為的三種組織形式：

| 形式 | 特徵 | 人類範例 |
|------|------|----------|
| **固定動作模式**（Fixed Action Patterns） | 極度刻板，無需環境回饋即可完成 | 嬰兒的哭泣、微笑、抓握 |
| **鏈式結構**（Chain Systems） | 一個行為結果終止該系統並促發下一個 | 蜜蜂覓食的視覺→嗅覺→觸覺序列 |
| **計畫層級**（Plan Hierarchy） | 最複雜靈活，整合多種類型子結構 | 人類早晨起床行為 |

人類依附系統從簡單的固定動作模式出發，逐漸發展為目標校正的計畫層級系統。

## 四、分離焦慮：創傷的核心機制

### 4.1 分離反應的三階段模型

鮑比整合羅伯森等人的觀察，提出兒童與母親分離時的典型三階段反應：

| 階段 | 行為特徵 | 持續時間 |
|------|----------|----------|
| **抗議**（Protest） | 強烈痛苦、大聲哭泣、發脾氣、拒絕替代者、時刻關注母親圖像或聲音 | 數小時至數天 |
| **絕望**（Despair） | 主動動作減少、間歇而單調的哭聲、退縮、哀傷狀態 | 數天至數週 |
| **疏離**（Detachment） | 接受照顧、對環境感興趣、對母親冷漠、以物體取代人際關係 | 可持續數月 |

**關鍵發現**：兒童 distress 的**根本變項**是母親的在場與否，而非陌生環境。母親陪伴時，陌生環境對孩子影響極微。

### 4.2 分離焦慮的生物學意義

鮑比從進化適應性環境的視角理解分離焦慮：

> 「在進化適應性環境中，與母親的分離使嬰兒暴露於捕食者風險。分離焦慮是一種保護機制，促使嬰兒盡快回到安全的照顧者身邊。」

這解釋了為何：

- 分離焦慮在6-8個月時達到高峰（與嬰兒運動能力增強同時）
- 母子關係越好，分離時焦慮越強烈（高品質關係意味著更穩定的安全基地）
- 遭受懲罰反而加強依附（逃避捕食者的本能反應）

### 4.3 機構成長的創傷效應

鮑比引用了大量關於機構養育（institutional care）對兒童影響的研究：

- 缺乏單一、主要照顧者的孩子，無法發展出穩定的依附關係
- 即使後來被收養，某些兒童（尤其1歲後才被收養者）仍無法形成親密關係
- 機構中成長的孩子社交行為出現嚴重紊亂

鮑比據此認為：

> 「兒童依附行為現有的組織模式會持續下去，隨年齡增長，修正難度越來越高。」

## 五、依附模式：個體差異與影響因素

### 5.1 安斯沃思的陌生人情境測驗

安斯沃思設計的標準化測驗為評估嬰兒依附模式提供了實證工具。測驗包含母親在場、離開、歸來等情境，持續約20分鐘，核心評價標準是依附關係的**安全程度**。

### 5.2 三種依附類型

| 類型 | 名稱 | 行為特徵 |
|------|------|----------|
| **B型** | 安全依附（Secure） | 將母親作為安全基地、自由探索環境、分離時適度沮喪但可被安撫、母親歸來時愉快迎接並尋求接觸 |
| **A型** | 焦慮迴避依附（Anxious-avoidant） | 主動迴避母親、對陌生人比對母親更友好、身體接觸時表現靠近/迴避衝突 |
| **C型** | 焦慮抗拒依附（Anxious-resistant） | 在尋求親近和抗拒之間搖擺、憤怒或消極、明顯表現出渴望更多接觸但無法被滿足 |

### 5.3 依附模式發展的影響因素

鮑比提出依附模式的發展是**嬰兒偏向**與**母親偏向**交互作用的結果：

**嬰兒偏向對母親的影響：**

- 出生後睡眠和哭泣時間的個體差異
- 男孩比女孩睡得更少、哭得更多
- 產前/產期神經生理損傷導致對噪音敏感、睡眠不安穩

**母親偏向對嬰兒的影響：**

- 母親在原生家庭的經歷（影響其照顧風格）
- 母親對嬰兒信號的敏感度
- 選擇干預時間的能力
- 從陪伴中獲得的共同喜悅

**有利於安全依附的母親行為指標：**

- 頻繁而持久的身體接觸（尤其出生後6個月內）
- 根據嬰兒節律選擇干預時間
- 提供規律的環境，使嬰兒感到行為會引發預期結果

### 5.4 依附模式的穩定性

多項追蹤研究顯示依附模式具有相當的穩定性：

| 研究者 | 發現 |
|--------|------|
| 康乃爾 | 81%的嬰兒在12和18個月時依附模式相同 |
| 沃特斯 | 96%的嬰兒在兩次測量中表現相同模式 |

**穩定性的條件：**

- 互動模式為雙方帶來的滿意度越高，越穩定
- 期望傾向於受到確認，造成正負循環
- 即使表面穩定，後來事件（事故、疾病、新生兒降臨）也可能帶來實質變化

## 六、安全基地：依附的核心功能

### 6.1 安全基地的概念

鮑比提出依附關係的核心功能是為兒童提供「安全基地」：

> 「安全基地是指孩子可以從那裡出發去探索周圍世界，確信母親會在那裡作為他歸來的基地，給他安慰和支持。」

這意味著：

- 母親在場時，兒童更有信心探索環境
- 母親不在或出現警覺信號時，依附行為急劇增加
- 探索行為與依附行為形成互補而非對立

### 6.2 安全基地與後續發展

安全依附的兒童在多方面展現優勢：

| 發展領域 | 安全依附兒童的表現 |
|----------|-------------------|
| **探索行為** | 玩耍時間更長、興趣更濃、微笑更多 |
| **社交能力** | 更願意合作，社交技巧更成熟 |
| **自我韌性** | 更高韌性，處於適度控制 |
| **人際信任** | 相信他人可給予幫助 |

到5-6歲時，安全依附兒童的這些能力已成為**孩子自身的能力**，而非依賴母親的在場。

### 6.3 安全基地與概念性角色替取

萊特（Right）的追蹤研究顯示：

- 4歲時概念性角色替取能力個體差異極大（9-37分）
- **母親的養育方式與兒童角色替取能力高度相關**
- 高分組母親會考慮兒童觀點並準備合理讓步
- 低分組母親堅持權威
- 遭到體罰的孩子，概念性角色替取能力特別低

這條證據鏈顯示：安全依附不僅影響情感發展，還深刻影響認知和社會理解能力的發展。

## 七、依附與人格發展：長程影響

### 7.1 正向循環 vs. 負向循環

鮑比提出依附模式與人格發展之間存在動態的自我強化循環：

**安全依附的正向循環：**

```
安全依附 → 價值感與信任 → 探索環境的自信 
         → 更豐富的認知與社交經驗 
         → 人格結構越來越結構化、具有彈性
```

**不安全依附的負向循環：**

```
不安全依附 → 自卑與不信任 → 應對環境的能力不足 
           → 經驗匱乏 
           → 人格結構彈性較低、缺乏控制、脆弱
```

### 7.2 早年依附經驗的持久影響

鮑比明確提出：

1. **早期形成的模式會持續存在**
2. **依附模式決定個體對後續負性事件（拒絕、分離、失落）的反應**
3. **不安全依附增加多種心理病理風險**

> 「在生命早年形成的內在工作模式……會傾向於持續下去，並為個體提供一個框架，用以解釋早年在家庭的經驗，以及解釋當前和未來與重要他人的關係。」

### 7.3 依附理論對臨床工作的啟示

鮑比將依附理論與臨床工作直接聯結：

- 分離焦慮應被視為「天性的反應」，而非病理性症狀
- 治療關係本身可以成為一種依附體驗，為患者提供「修正性情緒經驗」
- 理解患者的依附模式有助於預測其治療關係中的行為

## 八、理論整合：鮑比的知識創新

### 8.1 跨學科理論建構

鮑比的依附理論是一項高度跨學科的知識工程，其創新在於整合了多個原本隔閡的學科傳統：

| 學科傳統 | 鮑比的借鑒與創新 |
|----------|-----------------|
| **精神分析** | 保留早期經驗的核心地位，但去除能量模型 |
| **Ethology** | 採用本能行為的進化論框架 |
| **比較心理學** | 哈洛的絨猴實驗為「接觸舒適」提供決定性證據 |
| **控制論/系統論** | 以行為控制系統替代驅力概念 |
| **發展心理學** | 安斯沃思的實證分類系統提供操作化定義 |

### 8.2 對佛洛伊德理論的改造

鮑比對佛洛伊德的改造可以概括為以下核心替換：

| 佛洛伊德概念 | 鮑比的替代 |
|--------------|------------|
| 心靈能量（libido） | 行為控制系統 |
| 次級驅力（依附源於餵養） | 依附作為獨立本能系統 |
| 回溯法 | 前瞻性追蹤研究 |
| 創傷的象徵性解讀 | 分離作為真實創傷 |
| 心理決定論 | 演化適應性與個體發展的交互作用 |

### 8.3 鮑比的科學哲學立場

鮑比明確引用波普爾（Popper）的否證主義科學哲學，聲稱其理論立場：

- 理論應提出可被觀察證據檢驗的預測
- 未能通過檢驗的理論應被修正或拋棄
- 理論的價值在於其解釋力和預測力

這使依附理論不同於純粹的哲學或臨床猜測，而具有自然科學的驗證標準。

## 九、核心命題總結

鮑比在《依附》一書中的核心貢獻可以濃縮為以下命題：

### 命題一：依附是獨立的本能系統
依附行為不是由飢餓、口唇快感等次級驅力派生出來的，而是具有自身演化根源和適應功能的本能行為系統。

### 命題二：安全基地是依附的核心功能
母親作為安全基地，使嬰兒得以在確認情感保障的前提下探索環境、發展認知與社交能力。

### 命題三：分離是核心創傷來源
母子分離引發可預測的三階段反應（抗議→絕望→疏離），這些反應具有生物學適應基礎，但在現代環境中可能造成持久心理創傷。

### 命題四：依附模式具有個體差異和相對穩定性
兒童與照顧者互動的品質決定依附模式的類型（安全/焦慮迴避/焦慮抗拒），而這些模式具有跨時間的穩定性，並影響後續人格發展。

### 命題五：早期依附經驗形成持久的內在工作模式
早年形成的依附模式成為個體理解自己、他人與關係的認知框架，決定其對親密關係的期待和應對方式。

### 命題六：控制系統理論替代能量模型
人類行為應被理解為目標導向的調節系統，而非心靈能量的釋放與轉化；這使精神分析與現代生物學和控制論接軌。

## 十、延伸閱讀與理論影響

鮑比的依附理論對後續研究和臨床實踐产生了深远影響：

| 領域 | 代表人物/發展 |
|------|---------------|
| **依附類型擴展** | Main（1990）提出「組織型」vs.「失調型」依附 |
| **成人依附研究** | Hazan & Shaver（1987）將依附理論擴展至成人戀愛關係 |
| **依附與神經生物學** | 最新研究探索依附的神經機制 |
| **臨床應用** | 心智化基礎治療（MBT）、辨識情緒焦點治療（EFT）等 |

鮑比的工作不僅改造了精神分析的發展理論，還為理解人類情感、社會性和心理健康提供了新的概念框架，至今仍深刻影響著心理學、精神病學和兒童發展領域。
