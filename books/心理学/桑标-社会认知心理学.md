# 社会认知心理学

## 概述

社会认知心理学是认知心理学渗透到社会心理学并与之相结合而形成的一门新兴学科，形成于20世纪70年代末和80年代初。它以对人类自身和社会关系的认知为研究对象，探索人们如何理解他人和自己。其研究包括三个层次：对个人的认知、对人与人之间双边关系的认知、对群体内部或群体之间人们各种关系的认知。

社会认知心理学的核心观点是：**人是社会的**。它将认知心理学的新思想与传统社会心理学相结合，从考察人们如何表征社会概念转向考察人们如何表征社会概念并做出判断。

## 核心原则与基本观点

### 四大基本原则

1. **人是认知省事者**：人类信息加工能力有限，因此采用捷径和启发式策略，代价是可能缺乏准确性和深思熟虑。

2. **自动的与有意的思维**：动机和能力决定思维类型。缺乏动机或被其他任务占据时更可能发生自动思维；受到激励且具备能力时进行有意的、深思熟虑的思维。

3. **自尊的重要性**：自我评价是社会认知的关键。高自尊者知觉自己有能力、有价值；低自尊者可能缺乏动机进行有意思维。社会认同理论认为，人们追求认同良好群体以维护自尊。

4. **动机和情感**：从20世纪80年代起，“热认知”（动机和情感对认知的影响）重新获得关注，形成了认知、动机、情感综合研究的取向。

## 归因理论

归因是社会认知心理学的核心课题，指观察者根据外在行为对行动者或本人内部状态做出推论的过程。

### 主要归因理论

| 理论 | 代表人物 | 核心观点 |
|------|---------|----------|
| 通俗心理学 | 海德 | 人们有两种基本需要：形成一贯性理解的需要和控制环境的需要；使用“不变原则”进行因果归因 |
| 相应推论理论 | 琼斯、戴维斯 | 关注非共同效果、社会希望性、行动自愿性来确定行为与内在倾向的一致性 |
| 三维模型 | 凯利 | 实体的特异性、一致性反应、一贯性三维协变原则；折扣原则 |
| 自我知觉理论 | 贝姆 | 人们通过观察自己行为推断自己的态度、情感和内在状态 |
| 控制源理论 | 罗德 | 内在者相信自己控制强化事件；外在者认为由运气、他人控制，分为内在—外在控制源 |

### 韦纳的成就归因理论

韦纳在海德理论基础上增加了**稳定性**和**可控性**两个维度：

- **控制源**：内在（能力、努力）vs 外在（任务难度、运气）
- **稳定性**：稳定（能力、任务难度）vs 不稳定（努力、运气）
- **可控性**：可控（努力）vs 不可控（能力、运气）

归因后果影响期望、情绪和行为。稳定性决定未来期望，情绪反应由控制源和可控性决定。

### 归因偏差与错误

**基本归因错误**：高估行为者内在特质、低估情境因素的倾向。

**行动者-观察者偏差**：自我行为归因于情境，他人行为归因于内在特质。

**自我服务偏差**：成功归因于自己，失败归因于外在因素。

**一致性反应信息利用不足**：忽视基础概率信息。

## 心理控制

心理控制与归因密切相关，是自我概念和自尊的组成部分。

### 归因相关的控制

- **控制归因**：归因前提，假定行动者有能力完成行动
- **自由归因**：由情景条件决定知觉到的行动自由程度
- **责任归因**：因果归因、控制、自由归因的综合，产生赞赏或谴责

### 失去控制的反应

- **追求信息**：搜集信息、进行归因探索
- **应激反应**：高度生理激起，对其他任务关注减少
- **逆反**：控制期望受威胁时产生愤怒和恢复控制的努力
- **无助**：长期控制失败导致习得性无助

### 心理控制与消除紧张

| 控制类型 | 机制 |
|---------|------|
| 行为控制 | 采取步骤中止或改变事件 |
| 认知控制 | 重新构思消极事件或转移注意力 |
| 信息控制 | 获得关于事件程序和感觉的信息 |
| 决策控制 | 做出决定或从备择方案中选择 |
| 反省控制 | 分析已发生事件以提高控制感 |
| 次级控制 | 调整自我以适应环境 |

### 控制感的消极后果

- **控制幻想**：在机遇情景中高估控制
- **公正世界信念**：认为受害者应得不幸
- **过多控制**：信息超载导致适应性下降

## 社会图式

### 图式的定义与功能

社会图式是有关社会实体的知识结构，是从特定经验中抽象出来、用以指导新信息加工和储存信息提取的认知结构。

**核心功能**：简化现实，实现“认知经济”。

**基本假设**：知觉者主动构思现实，而非被动接受环境信息。

### 图式的类型

1. **人的图式**：关于他人特征和目标的认识，基于家族相似性原则运作
2. **自我图式**：关于自己个性、外表和行为的有组织的知识
3. **角色图式**：关于社会角色（如性别、职业）的规范和行为期望
4. **事件图式（脚本）**：关于熟悉情景中事件恰当次序的认知结构

### 图式的发展与激活

- 图式从具体到抽象发展
- 成熟图式更复杂、更有组织性
- 激活取决于**新近性**、**频率**、**观察目的**、**情感和动机**

### 图式的坚韧效应

即使面临矛盾证据，图式仍保持不变。通过监察判断过程或反驳图式可减轻该效应。

## 社会注意

### 社会刺激的突出性与鲜明性

- **突出性**：相对于背景的新异性、运动性、复杂性
- **显著性偏差**：过度依赖最易获得的信息
- **鲜明性**：情绪唤起性、具体性、感官丰富性

### 自我觉察

自我关注使个体将现实自我与理想自我进行比较，促进自我调节。

**自我调节控制过程**：比较→差异检测→行为调整→循环往复。

## 人际记忆

### 记忆的命题网络模型

- 信息以命题形式储存
- 联想性、有方向性、标定性
- 通过激活加强联结

### 记忆的编码类型

| 内容类型 | 编码形式 |
|---------|---------|
| 特质 | 命题编码 |
| 行为 | 时间编码（序列性） |
| 外貌 | 类比编码（表象） |
| 情感 | 情感编码 |

### 影响记忆的情景因素

- **任务目的**：印象形成、移情、自我参照比简单记忆产生更好记忆
- **先前情景**：启动效应影响编码和提取
- **材料组织**：熟悉度决定按个人还是按情境组织记忆

## 社会推论

### 启发式策略

| 启发式 | 功能 | 常见错误 |
|--------|------|----------|
| 代表性启发式 | 根据A与B的相似性判断概率 | 忽略先验概率、样本大小 |
| 可利用性启发式 | 根据提取容易程度估计频率 | 提取偏差、想象偏差 |
| 模拟启发式 | 想象假设情景的可能结果 | 对几近失误的错误解释 |
| 锚定和调整启发式 | 从参照点出发进行调整 | 调整不足 |

### 协变判断与错觉相关

- 人们难以正确使用所有四类证据（正面、反面及各自缺失）
- 存在**选择性取样**和**选择性回忆**
- **错觉相关**：高估期望变量之间的关联

### 规范模型与描述模型的对比

规范模型描述理想推论，描述模型解释实际推论。两者差异源于：
- 规范条件在现实中罕见
- 规范模型忽略内容和背景
- 反馈在推论中起关键作用

## 自我认知

### 自我知识的组织

- **自我复杂性**：高自我复杂性缓冲应激的消极影响
- **积极与消极面的组织**：分离vs整合的组织方式产生不同的情绪反应

### 自我差距与情绪

- **真实自我-理想自我差距**→沮丧、失望
- **真实自我-应该自我差距**→焦虑、罪感

### 自我知识与动机的关系

- **自我改进动机**：追求理想自我
- **自我提升动机**：维持积极自我评价
- **自我验证动机**：寻求证实自我概念的信息
- **自我调节**：控制指导自己行动

## 态度与社会认知

### 态度的三成分模型

- **认知成分**：对态度对象的心理印象
- **情感成分**：喜欢或厌恶的评价
- **行为倾向成分**：准备行动的心理状态

### 认知失调理论

当行为与态度不一致时产生失调体验，激发减少失调的动机。可通过改变态度、增加协调认知、减少失调认知重要性等方式减少失调。

### 认知反应分析

态度改变取决于沟通过程中产生的**认知反应**（支持性和反对性想法）。沟通者可信性、信息重复性、信息难度、听众介入程度都影响认知反应的质量。

## 行为与社会认知

### 认知-行为关系

两者关系受多种因素中介：
- 原型行为与认知更一致
- 测量水平匹配（具体↔具体，抽象↔抽象）
- 自我监察和自我意识的个别差异

### 印象管理

- **积极印象**：行为匹配、规范服从、奉承赞美、一致表现
- **模糊印象**：保持行为自由，避免被定型
- **自我障碍**：制造借口为失败预先准备

### 行为检验假设

- **自我实现预言**：错误期望导致符合期望的行为
- **假设确证偏差**：选择支持假设的信息

## 热认知：动机

### 动机性推论机制

动机影响判断的途径：
- 选择性接近支持性信息
- 以独特方式定义特性
- 选择性使用推论规则
- 努力投入程度差异

### 准确性目标vs指向性目标

- **准确性目标**：追求正确判断，导致更精细的信息加工
- **指向性目标**：追求期望结论，导致选择性信息处理

### 积极错觉

适度的积极错觉（如自我提升、控制感、乐观期望）具有适应性价值，可增强动机和坚持性。但可能导致忽视现实威胁。

## 热认知：情感

### 情绪的认知基础

**沙赫特-辛格理论**：情绪 = 激起 × 认知标签

情绪产生的两个成分：
1. 散漫的生理激起
2. 根据社会背景的认知解释

### 曼德勒的中断理论

- 环境输入中断目标或图式产生激起
- 激起强度与中断的目标复杂性成正比
- 认知解释决定情绪性质

### 情感对认知的影响

- **心境一致性效应**：良好心境导致积极判断和记忆
- **情感中断认知**：情绪促使重新评估当前目标

### 分离系统观点

情感和认知可能是独立运行的系统：
- 情感反应可能先于认知
- 简单暴露效应表明熟悉性可直接产生偏好
- 情感判断比认知判断更快、更直觉

## 延伸阅读

1. **《社会心理学》(戴维·迈尔斯著)**：西方最经典的社会心理学教材之一，系统介绍了社会认知的核心概念与研究，配合作者可获得更广泛的西方实证研究支持。

2. **《认知失调理论》(费斯汀格著，郑全全译)**：本书作者翻译的经典著作，深入理解态度改变的核心机制，为归因理论应用提供理论基础。

3. **《社会认知：理解自我与他人的科学》(Kunda, 1999)**：Social Cognition领域的权威专著，系统阐述了社会认知的计算模型和机制，适合希望深入研究该领域的读者。

4. **《不确定情境下的判断：启发式与偏差》(Kahneman, Tversky, Slovic)**：诺贝尔奖得主卡尼曼的经典论文集，系统介绍了本书中讨论的各种启发式策略和认知偏差的实证研究。
