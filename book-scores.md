## 评分标准

| 维度 | 说明 |
|------|------|
| **准确性** | 提炼是否准确反映原书主要观点和核心内容 |
| **清晰度** | 语言是否清晰易懂，表达是否流畅 |
| **结构性** | 是否有清晰的逻辑结构，内容是否按合理顺序组织 |
| **完整性** | 是否包含重要细节，是否遗漏关键信息 |
| **洞察性** | 是否提供新见解或独特视角，能否引发思考 |
| **综合分** | 加权综合评分（1-10分） |
