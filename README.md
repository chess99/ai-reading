# AI 阅读笔记

个人知识库，记录阅读笔记和思考。

## 📚 内容分类

- **个人成长**: 思维方式、习惯养成、认知升级
- **投资理财**: 价值投资、财务知识、投资智慧
- **商业管理**: 公司战略、市场营销、组织行为
- **心理学**: 认知心理、社会心理、决策科学
- **健康运动**: 运动科学、健康管理、生理知识
- **社会科学**: 经济学、社会学、人类历史
- **思维方式**: 系统思维、批判性思维

## 🚀 本地开发

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run docs:dev
```

访问 http://localhost:5173 查看效果。

### 构建静态网站

```bash
npm run docs:build
```

构建产物位于 `.vitepress/dist` 目录。

### 预览构建结果

```bash
npm run docs:preview
```

## 📝 写作规范

### 文件命名

文件名格式：`作者-书名.md`

例如：
- `丹尼尔·卡尼曼-思考，快与慢.md`
- `彼得·林奇-彼得·林奇的成功投资.md`

### Frontmatter

每篇笔记可以包含以下 frontmatter（可选）：

```yaml
---
title: 书名
author: 作者
category: 分类
tags: [标签1, 标签2]
---
```

构建时会自动从文件名提取信息并补充 frontmatter。

### 标签

在文章中使用 `#标签` 格式添加标签，构建时会自动提取。

例如：
```markdown
这本书讲述了 #价值投资 和 #长期主义 的重要性。
```

## 🛠️ 技术栈

- **VitePress**: 静态站点生成器
- **Vue 3**: UI 框架
- **Node.js**: 构建工具
- **GitHub Actions**: 自动部署
- **GitHub Pages**: 托管服务

## 📂 目录结构

```
AI阅读/
├── .github/workflows/    # GitHub Actions 配置
├── .vitepress/          # VitePress 配置
│   ├── config.mts       # 主配置文件
│   ├── theme/           # 自定义主题
│   └── scripts/         # 构建脚本
├── 个人成长/             # 笔记分类目录
├── 投资/
├── 商业管理/
└── ...
```

## 🔄 自动化

- **自动生成侧边栏**: 根据目录结构自动生成导航
- **自动提取标签**: 从文章内容中提取标签
- **自动补充元数据**: 从文件名提取作者和书名
- **自动部署**: 推送到 main 分支自动部署到 GitHub Pages
